settings.py 
1    ''' 
2    Settings 
3    ======== 
4     
5    .. versionadded:: 1.0.7 
6     
7    This module provides a complete and extensible framework for adding a 
8    Settings interface to your application. By default, the interface uses 
9    a :class:`SettingsWithSpinner`, which consists of a 
10   :class:`~kivy.uix.spinner.Spinner` (top) to switch between individual 
11   settings panels (bottom). See :ref:`differentlayouts` for some 
12   alternatives. 
13    
14   .. image:: images/settingswithspinner_kivy.jpg 
15       :align: center 
16    
17   A :class:`SettingsPanel` represents a group of configurable options. The 
18   :attr:`SettingsPanel.title` property is used by :class:`Settings` when a panel 
19   is added: it determines the name of the sidebar button. SettingsPanel controls 
20   a :class:`~kivy.config.ConfigParser` instance. 
21    
22   The panel can be automatically constructed from a JSON definition file: you 
23   describe the settings you want and corresponding sections/keys in the 
24   ConfigParser instance... and you're done! 
25    
26   Settings are also integrated into the :class:`~kivy.app.App` class. Use 
27   :meth:`Settings.add_kivy_panel` to configure the Kivy core settings in a panel. 
28    
29    
30   .. _settings_json: 
31    
32   Create a panel from JSON 
33   ------------------------ 
34    
35   To create a panel from a JSON-file, you need two things: 
36    
37       * a :class:`~kivy.config.ConfigParser` instance with default values 
38       * a JSON file 
39    
40   .. warning:: 
41    
42       The :class:`kivy.config.ConfigParser` is required. You cannot use the 
43       default ConfigParser from Python libraries. 
44    
45   You must create and handle the :class:`~kivy.config.ConfigParser` 
46   object. SettingsPanel will read the values from the associated 
47   ConfigParser instance. Make sure you have set default values (using 
48   :attr:`~kivy.config.ConfigParser.setdefaults`) for all the sections/keys 
49   in your JSON file! 
50    
51   The JSON file contains structured information to describe the available 
52   settings. Here is an example:: 
53    
54       [ 
55           { 
56               "type": "title", 
57               "title": "Windows" 
58           }, 
59           { 
60               "type": "bool", 
61               "title": "Fullscreen", 
62               "desc": "Set the window in windowed or fullscreen", 
63               "section": "graphics", 
64               "key": "fullscreen" 
65           } 
66       ] 
67    
68   Each element in the root list represents a setting that the user can configure. 
69   Only the "type" key is mandatory: an instance of the associated class will be 
70   created and used for the setting - other keys are assigned to corresponding 
71   properties of that class. 
72    
73       ============== ================================================= 
74       Type           Associated class 
75       -------------- ------------------------------------------------- 
76       title          :class:`SettingTitle` 
77       bool           :class:`SettingBoolean` 
78       numeric        :class:`SettingNumeric` 
79       options        :class:`SettingOptions` 
80       string         :class:`SettingString` 
81       path           :class:`SettingPath` 
82       ============== ================================================= 
83    
84       .. versionadded:: 1.1.0 
85           Added :attr:`SettingPath` type 
86    
87   In the JSON example above, the first element is of type "title". It will create 
88   a new instance of :class:`SettingTitle` and apply the rest of the key-value 
89   pairs to the properties of that class, i.e. "title": "Windows" sets the 
90   :attr:`~SettingsPanel.title` property of the panel to "Windows". 
91    
92   To load the JSON example to a :class:`Settings` instance, use the 
93   :meth:`Settings.add_json_panel` method. It will automatically instantiate a 
94   :class:`SettingsPanel` and add it to :class:`Settings`:: 
95    
96       from kivy.config import ConfigParser 
97    
98       config = ConfigParser() 
99       config.read('myconfig.ini') 
100   
101      s = Settings() 
102      s.add_json_panel('My custom panel', config, 'settings_custom.json') 
103      s.add_json_panel('Another panel', config, 'settings_test2.json') 
104   
105      # then use the s as a widget... 
106   
107   
108  .. _differentlayouts: 
109   
110  Different panel layouts 
111  ----------------------- 
112   
113  A kivy :class:`~kivy.app.App` can automatically create and display a 
114  :class:`Settings` instance. See the :attr:`~kivy.app.App.settings_cls` 
115  documentation for details on how to choose which settings class to 
116  display. 
117   
118  Several pre-built settings widgets are available. All except 
119  :class:`SettingsWithNoMenu` include close buttons triggering the 
120  on_close event. 
121   
122  - :class:`Settings`: Displays settings with a sidebar at the left to 
123    switch between json panels. 
124   
125  - :class:`SettingsWithSidebar`: A trivial subclass of 
126    :class:`Settings`. 
127   
128  - :class:`SettingsWithSpinner`: Displays settings with a spinner at 
129    the top, which can be used to switch between json panels. Uses 
130    :class:`InterfaceWithSpinner` as the 
131    :attr:`~Settings.interface_cls`. This is the default behavior from 
132    Kivy 1.8.0. 
133   
134  - :class:`SettingsWithTabbedPanel`: Displays json panels as individual 
135    tabs in a :class:`~kivy.uix.tabbedpanel.TabbedPanel`. Uses 
136    :class:`InterfaceWithTabbedPanel` as the :attr:`~Settings.interface_cls`. 
137   
138  - :class:`SettingsWithNoMenu`: Displays a single json panel, with no 
139    way to switch to other panels and no close button. This makes it 
140    impossible for the user to exit unless 
141    :meth:`~kivy.app.App.close_settings` is overridden with a different 
142    close trigger! Uses :class:`InterfaceWithNoMenu` as the 
143    :attr:`~Settings.interface_cls`. 
144   
145  You can construct your own settings panels with any layout you choose 
146  by setting :attr:`Settings.interface_cls`. This should be a widget 
147  that displays a json settings panel with some way to switch between 
148  panels. An instance will be automatically created by :class:`Settings`. 
149   
150  Interface widgets may be anything you like, but *must* have a method 
151  add_panel that receives newly created json settings panels for the 
152  interface to display. See the documentation for 
153  :class:`InterfaceWithSidebar` for more information. They may 
154  optionally dispatch an on_close event, for instance if a close button 
155  is clicked. This event is used by :class:`Settings` to trigger its own 
156  on_close event. 
157   
158  For a complete, working example, please see 
159  :file:`kivy/examples/settings/main.py`. 
160

198  class SettingItem(FloatLayout):
199      '''Base class for individual settings (within a panel). This class cannot 
200      be used directly; it is used for implementing the other setting classes. 
201      It builds a row with a title/description (left) and a setting control 
202      (right). 
203   
204      Look at :class:`SettingBoolean`, :class:`SettingNumeric` and 
205      :class:`SettingOptions` for usage examples. 
206   
207      :Events: 
208          `on_release` 
209              Fired when the item is touched and then released. 
210   
211      '''
212  
213      title = StringProperty('<No title set>')
214      '''Title of the setting, defaults to '<No title set>'. 
215   
216      :attr:`title` is a :class:`~kivy.properties.StringProperty` and defaults to 
217      '<No title set>'. 
218      '''
219  
220      desc = StringProperty(None, allownone=True)
221      '''Description of the setting, rendered on the line below the title. 
222   
223      :attr:`desc` is a :class:`~kivy.properties.StringProperty` and defaults to 
224      None. 
225      '''
226  
227      disabled = BooleanProperty(False)
228      '''Indicate if this setting is disabled. If True, all touches on the 
229      setting item will be discarded. 
230   
231      :attr:`disabled` is a :class:`~kivy.properties.BooleanProperty` and 
232      defaults to False. 
233      '''
234  
235      section = StringProperty(None)
236      '''Section of the token inside the :class:`~kivy.config.ConfigParser` 
237      instance. 
238   
239      :attr:`section` is a :class:`~kivy.properties.StringProperty` and defaults 
240      to None. 
241      '''
242  
243      key = StringProperty(None)
244      '''Key of the token inside the :attr:`section` in the 
245      :class:`~kivy.config.ConfigParser` instance. 
246   
247      :attr:`key` is a :class:`~kivy.properties.StringProperty` and defaults to 
248      None. 
249      '''
250  
251      value = ObjectProperty(None)
252      '''Value of the token according to the :class:`~kivy.config.ConfigParser` 
253      instance. Any change to this value will trigger a 
254      :meth:`Settings.on_config_change` event. 
255   
256      :attr:`value` is an :class:`~kivy.properties.ObjectProperty` and defaults 
257      to None. 
258      '''
259  
260      panel = ObjectProperty(None)
261      '''(internal) Reference to the SettingsPanel for this setting. You don't 
262      need to use it. 
263   
264      :attr:`panel` is an :class:`~kivy.properties.ObjectProperty` and defaults 
265      to None. 
266      '''
267  
268      content = ObjectProperty(None)
269      '''(internal) Reference to the widget that contains the real setting. 
270      As soon as the content object is set, any further call to add_widget will 
271      call the content.add_widget. This is automatically set. 
272   
273      :attr:`content` is an :class:`~kivy.properties.ObjectProperty` and defaults 
274      to None. 
275      '''
276  
277      selected_alpha = NumericProperty(0)
278      '''(internal) Float value from 0 to 1, used to animate the background when 
279      the user touches the item. 
280   
281      :attr:`selected_alpha` is a :class:`~kivy.properties.NumericProperty` and 
282      defaults to 0. 
283      '''
284  
285      __events__ = ('on_release', )
286  
287      def __init__(self, **kwargs):
288          super(SettingItem, self).__init__(**kwargs)
289          self.value = self.panel.get_value(self.section, self.key)
290  
291      def add_widget(self, *largs):
292          if self.content is None:
293              return super(SettingItem, self).add_widget(*largs)
294          return self.content.add_widget(*largs)
295  
296      def on_touch_down(self, touch):
297          if not self.collide_point(*touch.pos):
298              return
299          if self.disabled:
300              return
301          touch.grab(self)
302          self.selected_alpha = 1
303          return super(SettingItem, self).on_touch_down(touch)
304  
305      def on_touch_up(self, touch):
306          if touch.grab_current is self:
307              touch.ungrab(self)
308              self.dispatch('on_release')
309              Animation(selected_alpha=0, d=.25, t='out_quad').start(self)
310              return True
311          return super(SettingItem, self).on_touch_up(touch)
312  
313      def on_release(self):
314          pass
315  
316      def on_value(self, instance, value):
317          if not self.section or not self.key:
318              return
319          # get current value in config
320          panel = self.panel
321          if not isinstance(value, string_types):
322              value = str(value)
323          panel.set_value(self.section, self.key, value)

348  class SettingString(SettingItem):
349      '''Implementation of a string setting on top of a :class:`SettingItem`. 
350      It is visualized with a :class:`~kivy.uix.label.Label` widget that, when 
351      clicked, will open a :class:`~kivy.uix.popup.Popup` with a 
352      :class:`~kivy.uix.textinput.Textinput` so the user can enter a custom 
353      value. 
354      '''
355  
356      popup = ObjectProperty(None, allownone=True)
357      '''(internal) Used to store the current popup when it's shown. 
358   
359      :attr:`popup` is an :class:`~kivy.properties.ObjectProperty` and defaults 
360      to None. 
361      '''
362  
363      textinput = ObjectProperty(None)
364      '''(internal) Used to store the current textinput from the popup and 
365      to listen for changes. 
366   
367      :attr:`textinput` is an :class:`~kivy.properties.ObjectProperty` and 
368      defaults to None. 
369      '''
370  
371      def on_panel(self, instance, value):
372          if value is None:
373              return
374          self.fbind('on_release', self._create_popup)
375  
376      def _dismiss(self, *largs):
377          if self.textinput:
378              self.textinput.focus = False
379          if self.popup:
380              self.popup.dismiss()
381          self.popup = None
382  
383      def _validate(self, instance):
384          self._dismiss()
385          value = self.textinput.text.strip()
386          self.value = value
387  
388      def _create_popup(self, instance):
389          # create popup layout
390          content = BoxLayout(orientation='vertical', spacing='5dp')
391          popup_width = min(0.95 * Window.width, dp(500))
392          self.popup = popup = Popup(
393              title=self.title, content=content, size_hint=(None, None),
394              size=(popup_width, '250dp'))
395  
396          # create the textinput used for numeric input
397          self.textinput = textinput = TextInput(
398              text=self.value, font_size='24sp', multiline=False,
399              size_hint_y=None, height='42sp')
400          textinput.bind(on_text_validate=self._validate)
401          self.textinput = textinput
402  
403          # construct the content, widget are used as a spacer
404          content.add_widget(Widget())
405          content.add_widget(textinput)
406          content.add_widget(Widget())
407          content.add_widget(SettingSpacer())
408  
409          # 2 buttons are created for accept or cancel the current value
410          btnlayout = BoxLayout(size_hint_y=None, height='50dp', spacing='5dp')
411          btn = Button(text='Ok')
412          btn.bind(on_release=self._validate)
413          btnlayout.add_widget(btn)
414          btn = Button(text='Cancel')
415          btn.bind(on_release=self._dismiss)
416          btnlayout.add_widget(btn)
417          content.add_widget(btnlayout)
418  
419          # all done, open the popup !
420          popup.open()

423  class SettingPath(SettingItem):
424      '''Implementation of a Path setting on top of a :class:`SettingItem`. 
425      It is visualized with a :class:`~kivy.uix.label.Label` widget that, when 
426      clicked, will open a :class:`~kivy.uix.popup.Popup` with a 
427      :class:`~kivy.uix.filechooser.FileChooserListView` so the user can enter 
428      a custom value. 
429   
430      .. versionadded:: 1.1.0 
431      '''
432  
433      popup = ObjectProperty(None, allownone=True)
434      '''(internal) Used to store the current popup when it is shown. 
435   
436      :attr:`popup` is an :class:`~kivy.properties.ObjectProperty` and defaults 
437      to None. 
438      '''
439  
440      textinput = ObjectProperty(None)
441      '''(internal) Used to store the current textinput from the popup and 
442      to listen for changes. 
443   
444      :attr:`textinput` is an :class:`~kivy.properties.ObjectProperty` and 
445      defaults to None. 
446      '''
447  
448      show_hidden = BooleanProperty(False)
449      '''Whether to show 'hidden' filenames. What that means is 
450      operating-system-dependent. 
451   
452      :attr:`show_hidden` is an :class:`~kivy.properties.BooleanProperty` and 
453      defaults to False. 
454   
455      .. versionadded:: 1.10.0 
456      '''
457  
458      dirselect = BooleanProperty(True)
459      '''Whether to allow selection of directories. 
460   
461      :attr:`dirselect` is a :class:`~kivy.properties.BooleanProperty` and 
462      defaults to True. 
463   
464      .. versionadded:: 1.10.0 
465      '''
466  
467      def on_panel(self, instance, value):
468          if value is None:
469              return
470          self.fbind('on_release', self._create_popup)
471  
472      def _dismiss(self, *largs):
473          if self.textinput:
474              self.textinput.focus = False
475          if self.popup:
476              self.popup.dismiss()
477          self.popup = None
478  
479      def _validate(self, instance):
480          self._dismiss()
481          value = self.textinput.selection
482  
483          if not value:
484              return
485  
486          self.value = os.path.realpath(value[0])
487  
488      def _create_popup(self, instance):
489          # create popup layout
490          content = BoxLayout(orientation='vertical', spacing=5)
491          popup_width = min(0.95 * Window.width, dp(500))
492          self.popup = popup = Popup(
493              title=self.title, content=content, size_hint=(None, 0.9),
494              width=popup_width)
495  
496          # create the filechooser
497          initial_path = self.value or os.getcwd()
498          self.textinput = textinput = FileChooserListView(
499              path=initial_path, size_hint=(1, 1),
500              dirselect=self.dirselect, show_hidden=self.show_hidden)
501          textinput.bind(on_path=self._validate)
502  
503          # construct the content
504          content.add_widget(textinput)
505          content.add_widget(SettingSpacer())
506  
507          # 2 buttons are created for accept or cancel the current value
508          btnlayout = BoxLayout(size_hint_y=None, height='50dp', spacing='5dp')
509          btn = Button(text='Ok')
510          btn.bind(on_release=self._validate)
511          btnlayout.add_widget(btn)
512          btn = Button(text='Cancel')
513          btn.bind(on_release=self._dismiss)
514          btnlayout.add_widget(btn)
515          content.add_widget(btnlayout)
516  
517          # all done, open the popup !
518          popup.open()

521  class SettingNumeric(SettingString):
522      '''Implementation of a numeric setting on top of a :class:`SettingString`. 
523      It is visualized with a :class:`~kivy.uix.label.Label` widget that, when 
524      clicked, will open a :class:`~kivy.uix.popup.Popup` with a 
525      :class:`~kivy.uix.textinput.Textinput` so the user can enter a custom 
526      value. 
527      '''
528  
529      def _validate(self, instance):
530          # we know the type just by checking if there is a '.' in the original
531          # value
532          is_float = '.' in str(self.value)
533          self._dismiss()
534          try:
535              if is_float:
536                  self.value = text_type(float(self.textinput.text))
537              else:
538                  self.value = text_type(int(self.textinput.text))
539          except ValueError:
540              return

543  class SettingOptions(SettingItem):
544      '''Implementation of an option list on top of a :class:`SettingItem`. 
545      It is visualized with a :class:`~kivy.uix.label.Label` widget that, when 
546      clicked, will open a :class:`~kivy.uix.popup.Popup` with a 
547      list of options from which the user can select. 
548      '''
549  
550      options = ListProperty([])
551      '''List of all availables options. This must be a list of "string" items. 
552      Otherwise, it will crash. :) 
553   
554      :attr:`options` is a :class:`~kivy.properties.ListProperty` and defaults 
555      to []. 
556      '''
557  
558      popup = ObjectProperty(None, allownone=True)
559      '''(internal) Used to store the current popup when it is shown. 
560   
561      :attr:`popup` is an :class:`~kivy.properties.ObjectProperty` and defaults 
562      to None. 
563      '''
564  
565      def on_panel(self, instance, value):
566          if value is None:
567              return
568          self.fbind('on_release', self._create_popup)
569  
570      def _set_option(self, instance):
571          self.value = instance.text
572          self.popup.dismiss()
573  
574      def _create_popup(self, instance):
575          # create the popup
576          content = BoxLayout(orientation='vertical', spacing='5dp')
577          popup_width = min(0.95 * Window.width, dp(500))
578          self.popup = popup = Popup(
579              content=content, title=self.title, size_hint=(None, None),
580              size=(popup_width, '400dp'))
581          popup.height = len(self.options) * dp(55) + dp(150)
582  
583          # add all the options
584          content.add_widget(Widget(size_hint_y=None, height=1))
585          uid = str(self.uid)
586          for option in self.options:
587              state = 'down' if option == self.value else 'normal'
588              btn = ToggleButton(text=option, state=state, group=uid)
589              btn.bind(on_release=self._set_option)
590              content.add_widget(btn)
591  
592          # finally, add a cancel button to return on the previous panel
593          content.add_widget(SettingSpacer())
594          btn = Button(text='Cancel', size_hint_y=None, height=dp(50))
595          btn.bind(on_release=popup.dismiss)
596          content.add_widget(btn)
597  
598          # and open the popup !
599          popup.open()

611  class SettingsPanel(GridLayout):
612      '''This class is used to contruct panel settings, for use with a 
613      :class:`Settings` instance or subclass. 
614      '''
615  
616      title = StringProperty('Default title')
617      '''Title of the panel. The title will be reused by the :class:`Settings` in 
618      the sidebar. 
619      '''
620  
621      config = ObjectProperty(None, allownone=True)
622      '''A :class:`kivy.config.ConfigParser` instance. See module documentation 
623      for more information. 
624      '''
625  
626      settings = ObjectProperty(None)
627      '''A :class:`Settings` instance that will be used to fire the 
628      `on_config_change` event. 
629      '''
630  
631      def __init__(self, **kwargs):
632          kwargs.setdefault('cols', 1)
633          super(SettingsPanel, self).__init__(**kwargs)
634  
635      def on_config(self, instance, value):
636          if value is None:
637              return
638          if not isinstance(value, ConfigParser):
639              raise Exception('Invalid config object, you must use a'
640                              'kivy.config.ConfigParser, not another one !')
641  
642      def get_value(self, section, key):
643          '''Return the value of the section/key from the :attr:`config` 
644          ConfigParser instance. This function is used by :class:`SettingItem` to 
645          get the value for a given section/key. 
646   
647          If you don't want to use a ConfigParser instance, you might want to 
648          override this function. 
649          '''
650          config = self.config
651          if not config:
652              return
653          return config.get(section, key)
654  
655      def set_value(self, section, key, value):
656          current = self.get_value(section, key)
657          if current == value:
658              return
659          config = self.config
660          if config:
661              config.set(section, key, value)
662              config.write()
663          settings = self.settings
664          if settings:
665              settings.dispatch('on_config_change',
666                                config, section, key, value)

669  class InterfaceWithSidebar(BoxLayout):
670      '''The default Settings interface class. It displays a sidebar menu 
671      with names of available settings panels, which may be used to switch 
672      which one is currently displayed. 
673   
674      See :meth:`~InterfaceWithSidebar.add_panel` for information on the 
675      method you must implement if creating your own interface. 
676   
677      This class also dispatches an event 'on_close', which is triggered 
678      when the sidebar menu's close button is released. If creating your 
679      own interface widget, it should also dispatch such an event which 
680      will automatically be caught by :class:`Settings` and used to

789  class ContentPanel(ScrollView):
790      '''A class for displaying settings panels. It displays a single 
791      settings panel at a time, taking up the full size and shape of the 
792      ContentPanel. It is used by :class:`InterfaceWithSidebar` and 
793      :class:`InterfaceWithSpinner` to display settings. 
794   
795      '''
796  
797      panels = DictProperty({})
798      '''(internal) Stores a dictionary mapping settings panels to their uids. 
799   
800      :attr:`panels` is a :class:`~kivy.properties.DictProperty` and 
801      defaults to {}. 
802   
803      '''
804  
805      container = ObjectProperty()
806      '''(internal) A reference to the GridLayout that contains the 
807      settings panel. 
808   
809      :attr:`container` is an :class:`~kivy.properties.ObjectProperty` and 
810      defaults to None. 
811   
812      '''
813  
814      current_panel = ObjectProperty(None)
815      '''(internal) A reference to the current settings panel. 
816   
817      :attr:`current_panel` is an :class:`~kivy.properties.ObjectProperty` and 
818      defaults to None. 
819   
820      '''
821  
822      current_uid = NumericProperty(0)
823      '''(internal) A reference to the uid of the current settings panel. 
824   
825      :attr:`current_uid` is a 
826      :class:`~kivy.properties.NumericProperty` and defaults to 0. 
827   
828      '''
829  
830      def add_panel(self, panel, name, uid):
831          '''This method is used by Settings to add new panels for possible 
832          display. Any replacement for ContentPanel *must* implement 
833          this method. 
834   
835          :Parameters: 
836              `panel`: :class:`SettingsPanel` 
837                  It should be stored and displayed when requested. 
838              `name`: 
839                  The name of the panel as a string. It may be used to represent 
840                  the panel. 
841              `uid`: 
842                  A unique int identifying the panel. It should be stored and 
843                  used to identify panels when switching. 
844   
845          '''
846          self.panels[uid] = panel
847          if not self.current_uid:
848              self.current_uid = uid
849  
850      def on_current_uid(self, *args):
851          '''The uid of the currently displayed panel. Changing this will 
852          automatically change the displayed panel. 
853   
854          :Parameters: 
855              `uid`: 
856                  A panel uid. It should be used to retrieve and display 
857                  a settings panel that has previously been added with 
858                  :meth:`add_panel`. 
859   
860          '''
861          uid = self.current_uid
862          if uid in self.panels:
863              if self.current_panel is not None:
864                  self.remove_widget(self.current_panel)
865              new_panel = self.panels[uid]
866              self.add_widget(new_panel)
867              self.current_panel = new_panel
868              return True
869          return False  # New uid doesn't exist
870  
871      def add_widget(self, widget):
872          if self.container is None:
873              super(ContentPanel, self).add_widget(widget)
874          else:
875              self.container.add_widget(widget)
876  
877      def remove_widget(self, widget):
878          self.container.remove_widget(widget)

882  
883      '''Settings UI. Check module documentation for more information on how 
884      to use this class. 
885   
886      :Events: 
887          `on_config_change`: ConfigParser instance, section, key, value 
888              Fired when the section's key-value pair of a ConfigParser changes. 
889   
890              .. warning: 
891   
892                  value will be str/unicode type, regardless of the setting 
893                  type (numeric, boolean, etc) 
894          `on_close` 
895              Fired by the default panel when the Close button is pressed. 
896   
897          '''
898  
899      interface = ObjectProperty(None)
900      '''(internal) Reference to the widget that will contain, organise and 
901      display the panel configuration panel widgets. 
902   
903      :attr:`interface` is an :class:`~kivy.properties.ObjectProperty` and 
904      defaults to None. 
905   
906      '''
907  
908      interface_cls = ObjectProperty(InterfaceWithSidebar)
909      '''The widget class that will be used to display the graphical 
910      interface for the settings panel. By default, it displays one Settings 
911      panel at a time with a sidebar to switch between them. 
912   
913      :attr:`interface_cls` is an 
914      :class:`~kivy.properties.ObjectProperty` and defaults to

916   
917      .. versionchanged:: 1.8.0 
918          If you set a string, the :class:`~kivy.factory.Factory` will be used to

922  
923      __events__ = ('on_close', 'on_config_change')

925      def __init__(self, *args, **kargs):
926          self._types = {}
927          super(Settings, self).__init__(*args, **kargs)
928          self.add_interface()
929          self.register_type('string', SettingString)
930          self.register_type('bool', SettingBoolean)
931          self.register_type('numeric', SettingNumeric)
932          self.register_type('options', SettingOptions)

942          '''Register a new type that can be used in the JSON definition. 
943          '''

949      def add_interface(self):
950          '''(Internal) creates an instance of :attr:`Settings.interface_cls`, 
951          and sets it to :attr:`~Settings.interface`. When json panels are 
952          created, they will be added to this interface which will display them 
953          to the user. 
954          '''
955          cls = self.interface_cls
956          if isinstance(cls, string_types):
957              cls = Factory.get(cls)

959          self.interface = interface
960          self.add_widget(interface)
961          self.interface.bind(on_close=lambda j: self.dispatch('on_close'))

966      def add_json_panel(self, title, config, filename=None, data=None):
967          '''Create and add a new :class:`SettingsPanel` using the configuration 
968          `config` with the JSON definition `filename`. If `filename` is not set, 
969          then the JSON definition is read from the `data` parameter instead. 
970   
971          Check the :ref:`settings_json` section in the documentation for more 
972          information about JSON format and the usage of this function. 
973          '''
974          panel = self.create_json_panel(title, config, filename, data)
975          uid = panel.uid
976          if self.interface is not None:
977              self.interface.add_panel(panel, title, uid)
978  
979      def create_json_panel(self, title, config, filename=None, data=None):
980          '''Create new :class:`SettingsPanel`. 
981   
982          .. versionadded:: 1.5.0 
983   
984          Check the documentation of :meth:`add_json_panel` for more information. 
985          '''
986          if filename is None and data is None:
987              raise Exception('You must specify either the filename or data')
988          if filename is not None:
989              with open(filename, 'r') as fd:
990                  data = json.loads(fd.read())
991          else:
992              data = json.loads(data)
993          if type(data) != list:
994              raise ValueError('The first element must be a list')
995          panel = SettingsPanel(title=title, settings=self, config=config)
996  
997          for setting in data:
998              # determine the type and the class to use
999              if 'type' not in setting:
1000                 raise ValueError('One setting are missing the "type" element')
1001             ttype = setting['type']
1002             cls = self._types.get(ttype)
1003             if cls is None:
1004                 raise ValueError(
1005                     'No class registered to handle the <%s> type' %
1006                     setting['type'])
1007 
1008             # create a instance of the class, without the type attribute
1009             del setting['type']
1010             str_settings = {}
1011             for key, item in setting.items():
1012                 str_settings[str(key)] = item
1013 
1014             instance = cls(panel=panel, **str_settings)
1015 
1016             # instance created, add to the panel
1017             panel.add_widget(instance)
1018 
1019         return panel
