scrollview.py 
1    ''' 
2    ScrollView 
3    ========== 
4     
5    .. versionadded:: 1.0.4 
6     
7    The :class:`ScrollView` widget provides a scrollable/pannable viewport that is 
8    clipped at the scrollview's bounding box. 
9     
10    
11   Scrolling Behavior 
12   ------------------ 
13    
14   The ScrollView accepts only one child and applies a viewport/window to 
15   it according to the :attr:`~ScrollView.scroll_x` and 
16   :attr:`~ScrollView.scroll_y` properties. Touches are analyzed to 
17   determine if the user wants to scroll or control the child in some 
18   other manner: you cannot do both at the same time. To determine if 
19   interaction is a scrolling gesture, these properties are used: 
20    
21       - :attr:`~ScrollView.scroll_distance`: the minimum distance to travel, 
22         defaults to 20 pixels. 
23       - :attr:`~ScrollView.scroll_timeout`: the maximum time period, defaults 
24         to 55 milliseconds. 
25    
26   If a touch travels :attr:`~ScrollView.scroll_distance` pixels within the 
27   :attr:`~ScrollView.scroll_timeout` period, it is recognized as a scrolling 
28   gesture and translation (scroll/pan) will begin. If the timeout occurs, the 
29   touch down event is dispatched to the child instead (no translation). 
30    
31   The default value for those settings can be changed in the configuration file:: 
32    
33       [widgets] 
34       scroll_timeout = 250 
35       scroll_distance = 20 
36    
37   .. versionadded:: 1.1.1 
38    
39       ScrollView now animates scrolling in Y when a mousewheel is used. 
40    
41    
42   Limiting to the X or Y Axis 
43   --------------------------- 
44    
45   By default, the ScrollView allows scrolling along both the X and Y axes. You 
46   can explicitly disable scrolling on an axis by setting the 
47   :attr:`~ScrollView.do_scroll_x` or :attr:`~ScrollView.do_scroll_y` properties 
48   to False. 
49    
50    
51   Managing the Content Size and Position 
52   -------------------------------------- 
53    
54   The ScrollView manages the position of its children similarly to a 
55   :class:`~kivy.uix.relativelayout.RelativeLayout` but does not use the 
56   :attr:`~kivy.uix.widget.Widget.size_hint`. You must 
57   carefully specify the :attr:`~kivy.uix.widget.Widget.size` of your content to 
58   get the desired scroll/pan effect. 
59    
60   By default, the :attr:`~kivy.uix.widget.Widget.size_hint` is (1, 1), so the 
61   content size will fit your ScrollView 
62   exactly (you will have nothing to scroll). You must deactivate at least one of 
63   the size_hint instructions (x or y) of the child to enable scrolling. 
64   Setting :attr:`~kivy.uix.widget.Widget.size_hint_min` to not be None will 
65   also enable scrolling for that dimension when the :class:`ScrollView` is 
66   smaller than the minimum size. 
67    
68   To scroll a :class:`~kivy.uix.gridlayout.GridLayout` on it's Y-axis/vertically, 
69   set the child's width  to that of the ScrollView (size_hint_x=1), and set 
70   the size_hint_y property to None:: 
71    
72       from kivy.uix.gridlayout import GridLayout 
73       from kivy.uix.button import Button 
74       from kivy.uix.scrollview import ScrollView 
75       from kivy.core.window import Window 
76       from kivy.app import runTouchApp 
77    
78       layout = GridLayout(cols=1, spacing=10, size_hint_y=None) 
79       # Make sure the height is such that there is something to scroll. 
80       layout.bind(minimum_height=layout.setter('height')) 
81       for i in range(100): 
82           btn = Button(text=str(i), size_hint_y=None, height=40) 
83           layout.add_widget(btn) 
84       root = ScrollView(size_hint=(1, None), size=(Window.width, Window.height)) 
85       root.add_widget(layout) 
86    
87       runTouchApp(root) 
88    
89    
90   Kv Example:: 
91    
92       ScrollView: 
93           do_scroll_x: False 
94           do_scroll_y: True 
95    
96           Label: 
97               size_hint_y: None 
98               height: self.texture_size[1] 
99               text_size: self.width, None 
100              padding: 10, 10 
101              text: 
102                  'really some amazing text\\n' * 100 
103   
104  Overscroll Effects 
105  ------------------ 
106   
107  .. versionadded:: 1.7.0 
108   
109  When scrolling would exceed the bounds of the :class:`ScrollView`, it 
110  uses a :class:`~kivy.effects.scroll.ScrollEffect` to handle the 
111  overscroll. These effects can perform actions like bouncing back, 
112  changing opacity, or simply preventing scrolling beyond the normal 
113  boundaries. Note that complex effects may perform many computations, 
114  which can be slow on weaker hardware. 
115   
116  You can change what effect is being used by setting 
117  :attr:`~ScrollView.effect_cls` to any effect class. Current options 
118  include: 
119   
120      - :class:`~kivy.effects.scroll.ScrollEffect`: Does not allow 
121        scrolling beyond the :class:`ScrollView` boundaries. 
122      - :class:`~kivy.effects.dampedscroll.DampedScrollEffect`: The 
123        current default. Allows the user to scroll beyond the normal 
124        boundaries, but has the content spring back once the 
125        touch/click is released. 
126      - :class:`~kivy.effects.opacityscroll.OpacityScrollEffect`: Similar 
127        to the :class:`~kivy.effect.dampedscroll.DampedScrollEffect`, but 
128        also reduces opacity during overscroll. 
129   
130  You can also create your own scroll effect by subclassing one of these, 
131  then pass it as the :attr:`~ScrollView.effect_cls` in the same way. 
132   
133  Alternatively, you can set :attr:`~ScrollView.effect_x` and/or 
134  :attr:`~ScrollView.effect_y` to an *instance* of the effect you want to 
135  use. This will override the default effect set in 
136  :attr:`~ScrollView.effect_cls`. 
137   
138  All the effects are located in the :mod:`kivy.effects`. 
139

297      def _get_vbar(self):
298          # must return (y, height) in %
299          # calculate the viewport size / scrollview size %
300          if self._viewport is None:
301              return 0, 1.
302          vh = self._viewport.height
303          h = self.height
304          if vh < h or vh == 0:
305              return 0, 1.
306          ph = max(0.01, h / float(vh))
307          sy = min(1.0, max(0.0, self.scroll_y))
308          py = (1. - ph) * sy
309          return (py, ph)

326      def _get_hbar(self):
327          # must return (x, width) in %
328          # calculate the viewport size / scrollview size %
329          if self._viewport is None:
330              return 0, 1.
331          vw = self._viewport.width
332          w = self.width
333          if vw < w or vw == 0:
334              return 0, 1.
335          pw = max(0.01, w / float(vw))
336          sx = min(1.0, max(0.0, self.scroll_x))
337          px = (1. - pw) * sx
338          return (px, pw)

525      def __init__(self, **kwargs):
526          self._touch = None
527          self._trigger_update_from_scroll = Clock.create_trigger(
528              self.update_from_scroll, -1)
529          # create a specific canvas for the viewport
530          from kivy.graphics import PushMatrix, Translate, PopMatrix, Canvas
531          self.canvas_viewport = Canvas()
532          self.canvas = Canvas()
533          with self.canvas_viewport.before:
534              PushMatrix()
535              self.g_translate = Translate(0, 0)
536          with self.canvas_viewport.after:
537              PopMatrix()
538  
539          super(ScrollView, self).__init__(**kwargs)
540  
541          self.register_event_type('on_scroll_start')
542          self.register_event_type('on_scroll_move')
543          self.register_event_type('on_scroll_stop')
544  
545          # now add the viewport canvas to our canvas
546          self.canvas.add(self.canvas_viewport)
547  
548          effect_cls = self.effect_cls
549          if isinstance(effect_cls, string_types):
550              effect_cls = Factory.get(effect_cls)
551          if self.effect_x is None and effect_cls is not None:
552              self.effect_x = effect_cls(target_widget=self._viewport)
553          if self.effect_y is None and effect_cls is not None:
554              self.effect_y = effect_cls(target_widget=self._viewport)
555  
556          trigger_update_from_scroll = self._trigger_update_from_scroll
557          update_effect_widget = self._update_effect_widget
558          update_effect_x_bounds = self._update_effect_x_bounds
559          update_effect_y_bounds = self._update_effect_y_bounds
560          fbind = self.fbind
561          fbind('width', update_effect_x_bounds)
562          fbind('height', update_effect_y_bounds)
563          fbind('viewport_size', self._update_effect_bounds)
564          fbind('_viewport', update_effect_widget)
565          fbind('scroll_x', trigger_update_from_scroll)
566          fbind('scroll_y', trigger_update_from_scroll)
567          fbind('pos', trigger_update_from_scroll)
568          fbind('size', trigger_update_from_scroll)

570          trigger_update_from_scroll()
571          update_effect_widget()
572          update_effect_x_bounds()
573          update_effect_y_bounds()

593      def _update_effect_widget(self, *args):
594          if self.effect_x:
595              self.effect_x.target_widget = self._viewport
596          if self.effect_y:
597              self.effect_y.target_widget = self._viewport

599      def _update_effect_x_bounds(self, *args):
600          if not self._viewport or not self.effect_x:
601              return
602          scrollable_width = self.width - self.viewport_size[0]
603          self.effect_x.min = 0
604          self.effect_x.max = min(0, scrollable_width)
605          self.effect_x.value = scrollable_width * self.scroll_x
606  
607      def _update_effect_y_bounds(self, *args):
608          if not self._viewport or not self.effect_y:
609              return
610          scrollable_height = self.height - self.viewport_size[1]
611          self.effect_y.min = 0 if scrollable_height < 0 else scrollable_height
612          self.effect_y.max = scrollable_height
613          self.effect_y.value = self.effect_y.max * self.scroll_y
614  
615      def _update_effect_bounds(self, *args):
616          self._update_effect_x_bounds()
617          self._update_effect_y_bounds()
618  
619      def _update_effect_x(self, *args):
620          vp = self._viewport
621          if not vp or not self.effect_x:
622              return
623  
624          if self.effect_x.is_manual:
625              sw = vp.width - self._effect_x_start_width
626          else:
627              sw = vp.width - self.width
628          if sw < 1 and not (self.always_overscroll and self.do_scroll_x):
629              return
630          if sw != 0:
631              sx = self.effect_x.scroll / sw
632              self.scroll_x = -sx
633          self._trigger_update_from_scroll()
634  
635      def _update_effect_y(self, *args):
636          vp = self._viewport
637          if not vp or not self.effect_y:
638              return
639          if self.effect_y.is_manual:
640              sh = vp.height - self._effect_y_start_height
641          else:
642              sh = vp.height - self.height
643  
644          if sh < 1 and not (self.always_overscroll and self.do_scroll_y):
645              return
646          if sh != 0:
647              sy = self.effect_y.scroll / sh
648              self.scroll_y = -sy
649          self._trigger_update_from_scroll()

651      def to_local(self, x, y, **k):
652          tx, ty = self.g_translate.xy
653          return x - tx, y - ty

655      def to_parent(self, x, y, **k):
656          tx, ty = self.g_translate.xy
657          return x + tx, y + ty

659      def _apply_transform(self, m, pos=None):
660          tx, ty = self.g_translate.xy
661          m.translate(tx, ty, 0)
662          return super(ScrollView, self)._apply_transform(m, (0, 0))

664      def simulate_touch_down(self, touch):
665          # at this point the touch is in parent coords
666          touch.push()
667          touch.apply_transform_2d(self.to_local)
668          ret = super(ScrollView, self).on_touch_down(touch)
669          touch.pop()

673          if self.dispatch('on_scroll_start', touch):
674              self._touch = touch
675              touch.grab(self)
676              return True

678      def _touch_in_handle(self, pos, size, touch):
679          x, y = pos
680          width, height = size
681          return x <= touch.x <= x + width and y <= touch.y <= y + height

683      def on_scroll_start(self, touch, check_children=True):
684          if check_children:
685              touch.push()
686              touch.apply_transform_2d(self.to_local)
687              if self.dispatch_children('on_scroll_start', touch):
688                  touch.pop()
689                  return True
690              touch.pop()
691  
692          if not self.collide_point(*touch.pos):
693              touch.ud[self._get_uid('svavoid')] = True
694              return
695          if self.disabled:
696              return True
697          if self._touch or (not (self.do_scroll_x or self.do_scroll_y)):
698              return self.simulate_touch_down(touch)
699  
700          # handle mouse scrolling, only if the viewport size is bigger than the
701          # scrollview size, and if the user allowed to do it
702          vp = self._viewport
703          if not vp:
704              return True
705          scroll_type = self.scroll_type
706          ud = touch.ud
707          scroll_bar = 'bars' in scroll_type
708  
709          # check if touch is in bar_x(horizontal) or bar_y(vertical)
710          # width_enable_overscroll or vp.width > self.width
711          width_scrollable = self.always_overscroll or vp.width > self.width
712          height_scrollable = self.always_overscroll or vp.height > self.height
713  
714          d = {'bottom': touch.y - self.y - self.bar_margin,
715               'top': self.top - touch.y - self.bar_margin,
716               'left': touch.x - self.x - self.bar_margin,
717               'right': self.right - touch.x - self.bar_margin}
718  
719          ud['in_bar_x'] = (scroll_bar and width_scrollable and
720                            (0 <= d[self.bar_pos_x] <= self.bar_width))
721          ud['in_bar_y'] = (scroll_bar and height_scrollable and
722                            (0 <= d[self.bar_pos_y] <= self.bar_width))
723  
724          if 'button' in touch.profile and touch.button.startswith('scroll'):
725              btn = touch.button
726              m = self.scroll_wheel_distance
727              e = None
728  
729              if (
730                  (btn == 'scrolldown' and self.scroll_y >= 1)
731                  or (btn == 'scrollup' and self.scroll_y <= 0)
732                  or (btn == 'scrollleft' and self.scroll_x >= 1)
733                  or (btn == 'scrollright' and self.scroll_x <= 0)
734              ):
735                  return False
736  
737              if (
738                  self.effect_x
739                  and self.do_scroll_y
740                  and height_scrollable
741                  and btn in ('scrolldown', 'scrollup')
742              ):
743                  e = self.effect_x if ud['in_bar_x'] else self.effect_y
744  
745              elif (
746                  self.effect_y
747                  and self.do_scroll_x
748                  and width_scrollable
749                  and btn in ('scrollleft', 'scrollright')
750              ):
751                  e = self.effect_y if ud['in_bar_y'] else self.effect_x
752  
753              if e:
754                  # make sure the effect's value is synced to scroll value
755                  self._update_effect_bounds()
756                  if btn in ('scrolldown', 'scrollleft'):
757                      if self.smooth_scroll_end:
758                          e.velocity -= m * self.smooth_scroll_end
759                      else:
760                          if self.always_overscroll:
761                              e.value = e.value - m
762                          else:
763                              e.value = max(e.value - m, e.max)
764                          e.velocity = 0
765                  elif btn in ('scrollup', 'scrollright'):
766                      if self.smooth_scroll_end:
767                          e.velocity += m * self.smooth_scroll_end
768                      else:
769                          if self.always_overscroll:
770                              e.value = e.value + m
771                          else:
772                              e.value = min(e.value + m, e.min)
773                          e.velocity = 0
774                  touch.ud[self._get_uid('svavoid')] = True
775                  e.trigger_velocity_update()
776              return True
777  
778          in_bar = ud['in_bar_x'] or ud['in_bar_y']
779          if scroll_type == ['bars'] and not in_bar:
780              return self.simulate_touch_down(touch)
781  
782          if in_bar:
783              if (ud['in_bar_y'] and not
784                      self._touch_in_handle(
785                          self._handle_y_pos, self._handle_y_size, touch)):
786                  self.scroll_y = (touch.y - self.y) / self.height
787              elif (ud['in_bar_x'] and not
788                      self._touch_in_handle(
789                          self._handle_x_pos, self._handle_x_size, touch)):
790                  self.scroll_x = (touch.x - self.x) / self.width
791  
792          # no mouse scrolling, so the user is going to drag the scrollview with
793          # this touch.
794          self._touch = touch
795          uid = self._get_uid()
796  
797          ud[uid] = {
798              'mode': 'unknown',
799              'dx': 0,
800              'dy': 0,
801              'user_stopped': in_bar,
802              'frames': Clock.frames,
803              'time': touch.time_start,
804          }
805  
806          if (self.do_scroll_x and self.effect_x and not ud['in_bar_x']
807                  and not ud['in_bar_y']):
808              # make sure the effect's value is synced to scroll value
809              self._update_effect_bounds()
810  
811              self._effect_x_start_width = self.width
812              self.effect_x.start(touch.x)
813              self._scroll_x_mouse = self.scroll_x
814  
815          if (self.do_scroll_y and self.effect_y and not ud['in_bar_x']
816                  and not ud['in_bar_y']):
817              # make sure the effect's value is synced to scroll value
818              self._update_effect_bounds()
819  
820              self._effect_y_start_height = self.height
821              self.effect_y.start(touch.y)
822              self._scroll_y_mouse = self.scroll_y
823  
824          if not in_bar:
825              Clock.schedule_once(self._change_touch_mode,
826                                  self.scroll_timeout / 1000.)
827          return True
828  
829      def on_touch_move(self, touch):
830          if self._touch is not touch:
831              # don't pass on touch to children if outside the sv
832              if self.collide_point(*touch.pos):
833                  # touch is in parent
834                  touch.push()
835                  touch.apply_transform_2d(self.to_local)
836                  super(ScrollView, self).on_touch_move(touch)
837                  touch.pop()
838              return self._get_uid() in touch.ud
839          if touch.grab_current is not self:
840              return True
841  
842          if not any(isinstance(key, str) and key.startswith('sv.')
843                     for key in touch.ud):
844              # don't pass on touch to children if outside the sv
845              if self.collide_point(*touch.pos):
846                  # touch is in window coordinates
847                  touch.push()
848                  touch.apply_transform_2d(self.to_local)
849                  res = super(ScrollView, self).on_touch_move(touch)
850                  touch.pop()
851                  return res
852              return False
853  
854          touch.ud['sv.handled'] = {'x': False, 'y': False}
855          if self.dispatch('on_scroll_move', touch):
856              return True
857  
858      def on_scroll_move(self, touch):
859          if self._get_uid('svavoid') in touch.ud:
860              return False
861  
862          touch.push()
863          touch.apply_transform_2d(self.to_local)
864          if self.dispatch_children('on_scroll_move', touch):
865              touch.pop()
866              return True
867          touch.pop()
868  
869          rv = True
870  
871          # By default this touch can be used to defocus currently focused
872          # widget, like any touch outside of ScrollView.
873          touch.ud['sv.can_defocus'] = True
874  
875          uid = self._get_uid()
876          if uid not in touch.ud:
877              self._touch = False
878              return self.on_scroll_start(touch, False)
879          ud = touch.ud[uid]
880  
881          # check if the minimum distance has been travelled
882          if ud['mode'] == 'unknown':
883              if not (self.do_scroll_x or self.do_scroll_y):
884                  # touch is in parent, but _change expects window coords
885                  touch.push()
886                  touch.apply_transform_2d(self.to_local)
887                  touch.apply_transform_2d(self.to_window)
888                  self._change_touch_mode()
889                  touch.pop()
890                  return
891              ud['dx'] += abs(touch.dx)
892              ud['dy'] += abs(touch.dy)
893              if ((ud['dx'] > self.scroll_distance and self.do_scroll_x) or
894                      (ud['dy'] > self.scroll_distance and self.do_scroll_y)):
895                  ud['mode'] = 'scroll'
896  
897          if ud['mode'] == 'scroll':
898              if not touch.ud['sv.handled']['x'] and self.do_scroll_x \
899                      and self.effect_x:
900                  width = self.width
901                  if touch.ud.get('in_bar_x', False):
902                      if self.hbar[1] != 1:
903                          dx = touch.dx / float(width - width * self.hbar[1])
904                          self.scroll_x = min(max(self.scroll_x + dx, 0.), 1.)
905                          self._trigger_update_from_scroll()
906                  else:
907                      if self.scroll_type != ['bars']:
908                          self.effect_x.update(touch.x)
909                  if self.scroll_x < 0 or self.scroll_x > 1:
910                      rv = False
911                  else:
912                      touch.ud['sv.handled']['x'] = True
913                  # Touch resulted in scroll should not defocus focused widget
914                  touch.ud['sv.can_defocus'] = False
915              if not touch.ud['sv.handled']['y'] and self.do_scroll_y \
916                      and self.effect_y:
917                  height = self.height
918                  if touch.ud.get('in_bar_y', False):
919                      dy = touch.dy / float(height - height * self.vbar[1])
920                      self.scroll_y = min(max(self.scroll_y + dy, 0.), 1.)
921                      self._trigger_update_from_scroll()
922                  else:
923                      if self.scroll_type != ['bars']:
924                          self.effect_y.update(touch.y)
925                  if self.scroll_y < 0 or self.scroll_y > 1:
926                      rv = False
927                  else:
928                      touch.ud['sv.handled']['y'] = True
929                  # Touch resulted in scroll should not defocus focused widget
930                  touch.ud['sv.can_defocus'] = False
931              ud['dt'] = touch.time_update - ud['time']
932              ud['time'] = touch.time_update
933              ud['user_stopped'] = True
934          return rv
935  
936      def on_touch_up(self, touch):
937          uid = self._get_uid('svavoid')
938          if self._touch is not touch and uid not in touch.ud:
939              # don't pass on touch to children if outside the sv
940              if self.collide_point(*touch.pos):
941                  # touch is in parents
942                  touch.push()
943                  touch.apply_transform_2d(self.to_local)
944                  if super(ScrollView, self).on_touch_up(touch):
945                      touch.pop()
946                      return True
947                  touch.pop()
948              return False
949  
950          if self.dispatch('on_scroll_stop', touch):
951              touch.ungrab(self)
952              if not touch.ud.get('sv.can_defocus', True):
953                  # Focused widget should stay focused
954                  FocusBehavior.ignored_touch.append(touch)
955              return True

957      def on_scroll_stop(self, touch, check_children=True):
958          self._touch = None
959  
960          if check_children:
961              touch.push()
962              touch.apply_transform_2d(self.to_local)
963              if self.dispatch_children('on_scroll_stop', touch):
964                  touch.pop()
965                  return True
966              touch.pop()
967  
968          if self._get_uid('svavoid') in touch.ud:
969              return
970          if self._get_uid() not in touch.ud:
971              return False
972  
973          self._touch = None
974          uid = self._get_uid()
975          ud = touch.ud[uid]
976          if self.do_scroll_x and self.effect_x:
977              if not touch.ud.get('in_bar_x', False) and\
978                      self.scroll_type != ['bars']:
979                  self.effect_x.stop(touch.x)
980          if self.do_scroll_y and self.effect_y and\
981                  self.scroll_type != ['bars']:
982              if not touch.ud.get('in_bar_y', False):
983                  self.effect_y.stop(touch.y)
984          if ud['mode'] == 'unknown':
985              # we must do the click at least..
986              # only send the click if it was not a click to stop
987              # autoscrolling
988              if not ud['user_stopped']:
989                  self.simulate_touch_down(touch)
990              Clock.schedule_once(partial(self._do_touch_up, touch), .2)
991  
992          ev = self._update_effect_bounds_ev
993          if ev is None:
994              ev = self._update_effect_bounds_ev = Clock.create_trigger(
995                  self._update_effect_bounds)
996          ev()
997  
998          # if we do mouse scrolling, always accept it
999          if 'button' in touch.profile and touch.button.startswith('scroll'):
1000             return True
1001 
1002         return self._get_uid() in touch.ud

1004     def scroll_to(self, widget, padding=10, animate=True):
1005         '''Scrolls the viewport to ensure that the given widget is visible, 
1006         optionally with padding and animation. If animate is True (the 
1007         default), then the default animation parameters will be used. 
1008         Otherwise, it should be a dict containing arguments to pass to 
1009         :class:`~kivy.animation.Animation` constructor. 
1010  
1011         .. versionadded:: 1.9.1 
1012         '''
1013         if not self.parent:
1014             return
1015 
1016         # if _viewport is layout and has pending operation, reschedule
1017         if hasattr(self._viewport, 'do_layout'):
1018             if self._viewport._trigger_layout.is_triggered:
1019                 Clock.schedule_once(
1020                      lambda *dt: self.scroll_to(widget, padding, animate))
1021                 return
1022 
1023         if isinstance(padding, (int, float)):
1024             padding = (padding, padding)
1025 
1026         pos = self.parent.to_widget(*widget.to_window(*widget.pos))
1027         cor = self.parent.to_widget(*widget.to_window(widget.right,
1028                                                       widget.top))
1029 
1030         dx = dy = 0
1031 
1032         if pos[1] < self.y:
1033             dy = self.y - pos[1] + dp(padding[1])
1034         elif cor[1] > self.top:
1035             dy = self.top - cor[1] - dp(padding[1])
1036 
1037         if pos[0] < self.x:
1038             dx = self.x - pos[0] + dp(padding[0])
1039         elif cor[0] > self.right:
1040             dx = self.right - cor[0] - dp(padding[0])
1041 
1042         dsx, dsy = self.convert_distance_to_scroll(dx, dy)
1043         sxp = min(1, max(0, self.scroll_x - dsx))
1044         syp = min(1, max(0, self.scroll_y - dsy))
1045 
1046         if animate:
1047             if animate is True:
1048                 animate = {'d': 0.2, 't': 'out_quad'}
1049             Animation.stop_all(self, 'scroll_x', 'scroll_y')
1050             Animation(scroll_x=sxp, scroll_y=syp, **animate).start(self)
1051         else:
1052             self.scroll_x = sxp
1053             self.scroll_y = syp
1054 
1055     def convert_distance_to_scroll(self, dx, dy):
1056         '''Convert a distance in pixels to a scroll distance, depending on the 
1057         content size and the scrollview size. 
1058  
1059         The result will be a tuple of scroll distance that can be added to 
1060         :data:`scroll_x` and :data:`scroll_y` 
1061         '''
1062         if not self._viewport:
1063             return 0, 0
1064         vp = self._viewport
1065         if vp.width > self.width:
1066             sw = vp.width - self.width
1067             sx = dx / float(sw)
1068         else:
1069             sx = 0
1070         if vp.height > self.height:
1071             sh = vp.height - self.height
1072             sy = dy / float(sh)
1073         else:
1074             sy = 1
1075         return sx, sy

1077     def update_from_scroll(self, *largs):
1078         '''Force the reposition of the content, according to current value of 
1079         :attr:`scroll_x` and :attr:`scroll_y`. 
1080  
1081         This method is automatically called when one of the :attr:`scroll_x`, 
1082         :attr:`scroll_y`, :attr:`pos` or :attr:`size` properties change, or 
1083         if the size of the content changes. 
1084         '''
1085         if not self._viewport:
1086             self.g_translate.xy = self.pos
1087             return
1088         vp = self._viewport
1089 
1090         # update from size_hint
1091         if vp.size_hint_x is not None:
1092             w = vp.size_hint_x * self.width
1093             if vp.size_hint_min_x is not None:
1094                 w = max(w, vp.size_hint_min_x)
1095             if vp.size_hint_max_x is not None:
1096                 w = min(w, vp.size_hint_max_x)
1097             vp.width = w
1098 
1099         if vp.size_hint_y is not None:
1100             h = vp.size_hint_y * self.height
1101             if vp.size_hint_min_y is not None:
1102                 h = max(h, vp.size_hint_min_y)
1103             if vp.size_hint_max_y is not None:
1104                 h = min(h, vp.size_hint_max_y)
1105             vp.height = h
1106 
1107         if vp.width > self.width or self.always_overscroll:
1108             sw = vp.width - self.width
1109             x = self.x - self.scroll_x * sw
1110         else:
1111             x = self.x
1112 
1113         if vp.height > self.height or self.always_overscroll:
1114             sh = vp.height - self.height
1115             y = self.y - self.scroll_y * sh
1116         else:
1117             y = self.top - vp.height
1118 
1119         # from 1.8.0, we now use a matrix by default, instead of moving the
1120         # widget position behind. We set it here, but it will be a no-op most
1121         # of the time.
1122         vp.pos = 0, 0
1123         self.g_translate.xy = x, y
1124 
1125         # New in 1.2.0, show bar when scrolling happens and (changed in 1.9.0)
1126         # fade to bar_inactive_color when no scroll is happening.
1127         ev = self._bind_inactive_bar_color_ev
1128         if ev is None:
1129             ev = self._bind_inactive_bar_color_ev = Clock.create_trigger(
1130                 self._bind_inactive_bar_color, .5)
1131         self.funbind('bar_inactive_color', self._change_bar_color)
1132         Animation.stop_all(self, '_bar_color')
1133         self.fbind('bar_color', self._change_bar_color)
1134         self._bar_color = self.bar_color
1135         ev()

1140         Animation(
1141             _bar_color=self.bar_inactive_color,
1142             d=.5, t='out_quart').start(self)

1173     def _change_touch_mode(self, *largs):
1174         if not self._touch:
1175             return
1176         uid = self._get_uid()
1177         touch = self._touch
1178         if uid not in touch.ud:
1179             self._touch = False
1180             return
1181         ud = touch.ud[uid]
1182         if ud['mode'] != 'unknown' or ud['user_stopped']:
1183             return
1184         diff_frames = Clock.frames - ud['frames']
1185 
1186         # in order to be able to scroll on very slow devices, let at least 3
1187         # frames displayed to accumulate some velocity. And then, change the
1188         # touch mode. Otherwise, we might never be able to compute velocity,
1189         # and no way to scroll it. See #1464 and #1499
1190         if diff_frames < 3:
1191             Clock.schedule_once(self._change_touch_mode, 0)
1192             return
1193 
1194         if self.do_scroll_x and self.effect_x:
1195             self.effect_x.cancel()
1196         if self.do_scroll_y and self.effect_y:
1197             self.effect_y.cancel()
1198         # XXX the next line was in the condition. But this stop
1199         # the possibility to "drag" an object out of the scrollview in the
1200         # non-used direction: if you have an horizontal scrollview, a
1201         # vertical gesture will not "stop" the scroll view to look for an
1202         # horizontal gesture, until the timeout is done.
1203         # and touch.dx + touch.dy == 0:
1204         touch.ungrab(self)
1205         self._touch = None
1206         # touch is in window coords
1207         touch.push()
1208         touch.apply_transform_2d(self.to_widget)
1209         touch.apply_transform_2d(self.to_parent)
1210         self.simulate_touch_down(touch)
1211         touch.pop()
1212         return

1214     def _do_touch_up(self, touch, *largs):
1215         # touch is in window coords
1216         touch.push()
1217         touch.apply_transform_2d(self.to_widget)
1218         super(ScrollView, self).on_touch_up(touch)
1219         touch.pop()
1220         # don't forget about grab event!
1221         for x in touch.grab_list[:]:
1222             touch.grab_list.remove(x)
1223             x = x()
1224             if not x:
1225                 continue
1226             touch.grab_current = x
1227             # touch is in window coords
1228             touch.push()
1229             touch.apply_transform_2d(self.to_widget)
1230             super(ScrollView, self).on_touch_up(touch)
1231             touch.pop()
1232         touch.grab_current = None
