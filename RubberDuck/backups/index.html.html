13   <H2>Python and COM</H2>
14   <H3>Introduction</H3>
15   <P>Python has an excellent interface to COM (also known variously as OLE2, ActiveX, etc).</P>
16   <P>The Python COM package can be used to interface to almost any COM program (such as the MS-Office suite), write servers that can be hosted by any COM client (such as Visual Basic or C++), and has even been used to provide the core ActiveX Scripting Support. </P>
