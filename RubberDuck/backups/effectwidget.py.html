1    ''' 
2    EffectWidget 
3    ============ 
4     
5    .. versionadded:: 1.9.0 
6     
7    The :class:`EffectWidget` is able to apply a variety of fancy 
8    graphical effects to 
9    its children. It works by rendering to a series of 
10   :class:`~kivy.graphics.Fbo` instances with custom opengl fragment shaders. 
11   As such, effects can freely do almost anything, from inverting the 
12   colors of the widget, to anti-aliasing, to emulating the appearance of a 
13   crt monitor! 
14    
15   .. warning:: 
16       This code is still experimental, and its API is subject to change in a 
17       future version. 
18    
19   The basic usage is as follows:: 
20    
21       w = EffectWidget() 
22       w.add_widget(Button(text='Hello!') 
23       w.effects = [InvertEffect(), HorizontalBlurEffect(size=2.0)] 
24    
25   The equivalent in kv would be:: 
26    
27       #: import ew kivy.uix.effectwidget 
28       EffectWidget: 
29           effects: ew.InvertEffect(), ew.HorizontalBlurEffect(size=2.0) 
30           Button: 
31               text: 'Hello!' 
32    
33   The effects can be a list of effects of any length, and they will be 
34   applied sequentially. 
35    
36   The module comes with a range of prebuilt effects, but the interface 
37   is designed to make it easy to create your own. Instead of writing a 
38   full glsl shader, you provide a single function that takes 
39   some inputs based on the screen (current pixel color, current widget 
40   texture etc.). See the sections below for more information. 
41    
42   Usage Guidelines 
43   ---------------- 
44    
45   It is not efficient to resize an :class:`EffectWidget`, as 
46   the :class:`~kivy.graphics.Fbo` is recreated on each resize event. 
47   If you need to resize frequently, consider doing things a different 
48   way. 
49    
50   Although some effects have adjustable parameters, it is 
51   *not* efficient to animate these, as the entire 
52   shader is reconstructed every time. You should use glsl 
53   uniform variables instead. The :class:`AdvancedEffectBase` 
54   may make this easier. 
55    
56   .. note:: The :class:`EffectWidget` *cannot* draw outside its own 
57             widget area (pos -> pos + size). Any child widgets 
58             overlapping the boundary will be cut off at this point. 
59    
60   Provided Effects 
61   ---------------- 
62    
63   The module comes with several pre-written effects. Some have 
64   adjustable properties (e.g. blur radius). Please see the individual 
65   effect documentation for more details. 
66    
67   - :class:`MonochromeEffect` - makes the widget grayscale. 
68   - :class:`InvertEffect` - inverts the widget colors. 
69   - :class:`ChannelMixEffect` - swaps color channels. 
70   - :class:`ScanlinesEffect` - displays flickering scanlines. 
71   - :class:`PixelateEffect` - pixelates the image. 
72   - :class:`HorizontalBlurEffect` - Gaussuan blurs horizontally. 
73   - :class:`VerticalBlurEffect` - Gaussuan blurs vertically. 
74   - :class:`FXAAEffect` - applies a very basic anti-aliasing. 
75    
76   Creating Effects 
77   ---------------- 
78    
79   Effects are designed to make it easy to create and use your own 
80   transformations. You do this by creating and using an instance of 
81   :class:`EffectBase` with your own custom :attr:`EffectBase.glsl` 
82   property. 
83    
84   The glsl property is a string representing part of a glsl fragment 
85   shader. You can include as many functions as you like (the string 
86   is simply spliced into the whole shader), but it 
87   must implement a function :code:`effect` as below:: 
88    
89       vec4 effect(vec4 color, sampler2D texture, vec2 tex_coords, vec2 coords) 
90       { 
91           // ... your code here 
92           return something;  // must be a vec4 representing the new color 
93       } 
94    
95   The full shader will calculate the normal pixel color at each point, 
96   then call your :code:`effect` function to transform it. The 
97   parameters are: 
98    
99   - **color**: The normal color of the current pixel (i.e. texture 
100    sampled at tex_coords). 
101  - **texture**: The texture containing the widget's normal background. 
102  - **tex_coords**: The normal texture_coords used to access texture. 
103  - **coords**: The pixel indices of the current pixel. 
104   
105  The shader code also has access to two useful uniform variables, 
106  :code:`time` containing the time (in seconds) since the program start, 
107  and :code:`resolution` containing the shape (x pixels, y pixels) of 
108  the widget. 
109   
110  For instance, the following simple string (taken from the `InvertEffect`) 
111  would invert the input color but set alpha to 1.0:: 
112   
113      vec4 effect(vec4 color, sampler2D texture, vec2 tex_coords, vec2 coords) 
114      { 
115          return vec4(1.0 - color.xyz, 1.0); 
116      } 
117   
118  You can also set the glsl by automatically loading the string from a 
119  file, simply set the :attr:`EffectBase.source` property of an effect. 
120

300  effect_fxaa = ''' 
301  vec4 effect( vec4 color, sampler2D buf0, vec2 texCoords, vec2 coords) 
302  { 
303   
304      vec2 frameBufSize = resolution; 
305   
306      float FXAA_SPAN_MAX = 8.0; 
307      float FXAA_REDUCE_MUL = 1.0/8.0; 
308      float FXAA_REDUCE_MIN = 1.0/128.0; 
309   
310      vec3 rgbNW=texture2D(buf0,texCoords+(vec2(-1.0,-1.0)/frameBufSize)).xyz; 
311      vec3 rgbNE=texture2D(buf0,texCoords+(vec2(1.0,-1.0)/frameBufSize)).xyz; 
312      vec3 rgbSW=texture2D(buf0,texCoords+(vec2(-1.0,1.0)/frameBufSize)).xyz; 
313      vec3 rgbSE=texture2D(buf0,texCoords+(vec2(1.0,1.0)/frameBufSize)).xyz; 
314      vec3 rgbM=texture2D(buf0,texCoords).xyz; 
315   
316      vec3 luma=vec3(0.299, 0.587, 0.114); 
317      float lumaNW = dot(rgbNW, luma); 
318      float lumaNE = dot(rgbNE, luma); 
319      float lumaSW = dot(rgbSW, luma); 
320      float lumaSE = dot(rgbSE, luma); 
321      float lumaM  = dot(rgbM, luma); 
322   
323      float lumaMin = min(lumaM, min(min(lumaNW, lumaNE), min(lumaSW, lumaSE))); 
324      float lumaMax = max(lumaM, max(max(lumaNW, lumaNE), max(lumaSW, lumaSE))); 
325   
326      vec2 dir; 
327      dir.x = -((lumaNW + lumaNE) - (lumaSW + lumaSE)); 
328      dir.y =  ((lumaNW + lumaSW) - (lumaNE + lumaSE)); 
329   
330      float dirReduce = max( 
331          (lumaNW + lumaNE + lumaSW + lumaSE) * (0.25 * FXAA_REDUCE_MUL), 
332          FXAA_REDUCE_MIN); 
333   
334      float rcpDirMin = 1.0/(min(abs(dir.x), abs(dir.y)) + dirReduce); 
335   
336      dir = min(vec2(FXAA_SPAN_MAX, FXAA_SPAN_MAX), 
337            max(vec2(-FXAA_SPAN_MAX, -FXAA_SPAN_MAX), 
338            dir * rcpDirMin)) / frameBufSize; 
339   
340      vec3 rgbA = (1.0/2.0) * ( 
341          texture2D(buf0, texCoords.xy + dir * (1.0/3.0 - 0.5)).xyz + 
342          texture2D(buf0, texCoords.xy + dir * (2.0/3.0 - 0.5)).xyz); 
343      vec3 rgbB = rgbA * (1.0/2.0) + (1.0/4.0) * ( 
344          texture2D(buf0, texCoords.xy + dir * (0.0/3.0 - 0.5)).xyz + 
345          texture2D(buf0, texCoords.xy + dir * (3.0/3.0 - 0.5)).xyz); 
346      float lumaB = dot(rgbB, luma); 
347   
348      vec4 return_color; 
349      if((lumaB < lumaMin) || (lumaB > lumaMax)){ 
350          return_color = vec4(rgbA, color.w); 
351      }else{ 
352          return_color = vec4(rgbB, color.w); 
353      } 
354   
355      return return_color; 
356  } 
357  '''

360  class EffectBase(EventDispatcher):
361      '''The base class for GLSL effects. It simply returns its input. 
362   
363      See the module documentation for more details. 
364   
365      '''
366  
367      glsl = StringProperty(effect_trivial)
368      '''The glsl string defining your effect function. See the 
369      module documentation for more details. 
370   
371      :attr:`glsl` is a :class:`~kivy.properties.StringProperty` and 
372      defaults to 
373      a trivial effect that returns its input. 
374      '''
375  
376      source = StringProperty('')
377      '''The (optional) filename from which to load the :attr:`glsl` 
378      string. 
379   
380      :attr:`source` is a :class:`~kivy.properties.StringProperty` and 
381      defaults to ''. 
382      '''
383  
384      fbo = ObjectProperty(None, allownone=True)
385      '''The fbo currently using this effect. The :class:`EffectBase` 
386      automatically handles this. 
387   
388      :attr:`fbo` is an :class:`~kivy.properties.ObjectProperty` and 
389      defaults to None. 
390      '''
391  
392      def __init__(self, *args, **kwargs):
393          super(EffectBase, self).__init__(*args, **kwargs)
394          fbind = self.fbind
395          fbo_shader = self.set_fbo_shader
396          fbind('fbo', fbo_shader)
397          fbind('glsl', fbo_shader)
398          fbind('source', self._load_from_source)
399  
400      def set_fbo_shader(self, *args):
401          '''Sets the :class:`~kivy.graphics.Fbo`'s shader by splicing 
402          the :attr:`glsl` string into a full fragment shader. 
403   
404          The full shader is made up of :code:`shader_header + 
405          shader_uniforms + self.glsl + shader_footer_effect`. 
406          '''
407          if self.fbo is None:
408              return
409          self.fbo.set_fs(shader_header + shader_uniforms + self.glsl +
410                          shader_footer_effect)
411  
412      def _load_from_source(self, *args):
413          '''(internal) Loads the glsl string from a source file.'''
414          source = self.source
415          if not source:
416              return
417          filename = resource_find(source)
418          if filename is None:
419              return Logger.error('Error reading file {filename}'.
420                                  format(filename=source))
421          with open(filename) as fileh:
422              self.glsl = fileh.read()

425  class AdvancedEffectBase(EffectBase):
426      '''An :class:`EffectBase` with additional behavior to easily 
427      set and update uniform variables in your shader. 
428   
429      This class is provided for convenience when implementing your own 
430      effects: it is not used by any of those provided with Kivy. 
431   
432      In addition to your base glsl string that must be provided as 
433      normal, the :class:`AdvancedEffectBase` has an extra property 
434      :attr:`uniforms`, a dictionary of name-value pairs. Whenever 
435      a value is changed, the new value for the uniform variable is 
436      uploaded to the shader. 
437   
438      You must still manually declare your uniform variables at the top 
439      of your glsl string. 
440      '''
441  
442      uniforms = DictProperty({})
443      '''A dictionary of uniform variable names and their values. These 
444      are automatically uploaded to the :attr:`fbo` shader if appropriate. 
445   
446      uniforms is a :class:`~kivy.properties.DictProperty` and 
447      defaults to {}. 
448      '''
449  
450      def __init__(self, *args, **kwargs):
451          super(AdvancedEffectBase, self).__init__(*args, **kwargs)
452          self.fbind('uniforms', self._update_uniforms)
453  
454      def _update_uniforms(self, *args):
455          if self.fbo is None:
456              return
457          for key, value in self.uniforms.items():
458              self.fbo[key] = value
459  
460      def set_fbo_shader(self, *args):
461          super(AdvancedEffectBase, self).set_fbo_shader(*args)
462          self._update_uniforms()

584  class EffectFbo(Fbo):
585      '''An :class:`~kivy.graphics.Fbo` with extra functionality that allows 
586      attempts to set a new shader. See :meth:`set_fs`. 
587      '''
588      def __init__(self, *args, **kwargs):
589          kwargs.setdefault("with_stencilbuffer", True)
590          super(EffectFbo, self).__init__(*args, **kwargs)
591          self.texture_rectangle = None
592  
593      def set_fs(self, value):
594          '''Attempt to set the fragment shader to the given value. 
595          If setting the shader fails, the existing one is preserved and an 
596          exception is raised. 
597          '''
598          shader = self.shader
599          old_value = shader.fs
600          shader.fs = value
601          if not shader.success:
602              shader.fs = old_value
603              raise Exception('Setting new shader failed.')

619      '''
620  
621      texture = ObjectProperty(None)
622      '''The output texture of the final :class:`~kivy.graphics.Fbo` after 
623      all effects have been applied. 
624   
625      texture is an :class:`~kivy.properties.ObjectProperty` and defaults 
626      to None. 
627      '''
628  
629      effects = ListProperty([])
630      '''List of all the effects to be applied. These should all be 
631      instances or subclasses of :class:`EffectBase`. 
632   
633      effects is a :class:`ListProperty` and defaults to []. 
634      '''
635  
636      fbo_list = ListProperty([])
637      '''(internal) List of all the fbos that are being used to apply 
638      the effects. 
639   
640      fbo_list is a :class:`ListProperty` and defaults to []. 
641      '''
642  
643      _bound_effects = ListProperty([])
644      '''(internal) List of effect classes that have been given an fbo to 
645      manage. This is necessary so that the fbo can be removed if the 
646      effect is no longer in use. 
647   
648      _bound_effects is a :class:`ListProperty` and defaults to []. 
649      '''
650  
651      def __init__(self, **kwargs):
652          # Make sure opengl context exists
653          EventLoop.ensure_window()
654  
655          self.canvas = RenderContext(use_parent_projection=True,
656                                      use_parent_modelview=True)
657  
658          with self.canvas:
659              self.fbo = Fbo(size=self.size)
660  
661          with self.fbo.before:
662              PushMatrix()
663          with self.fbo:
664              ClearColor(0, 0, 0, 0)
665              ClearBuffers()
666              self._background_color = Color(*self.background_color)
667              self.fbo_rectangle = Rectangle(size=self.size)
668          with self.fbo.after:
669              PopMatrix()
670  
671          super(EffectWidget, self).__init__(**kwargs)
672  
673          Clock.schedule_interval(self._update_glsl, 0)
674  
675          fbind = self.fbind
676          fbo_setup = self.refresh_fbo_setup
677          fbind('size', fbo_setup)
678          fbind('effects', fbo_setup)
679          fbind('background_color', self._refresh_background_color)
680  
681          self.refresh_fbo_setup()
682          self._refresh_background_color()  # In case thi was changed in kwargs
683  
684      def _refresh_background_color(self, *args):
685          self._background_color.rgba = self.background_color
686  
687      def _update_glsl(self, *largs):
688          '''(internal) Passes new time and resolution uniform 
689          variables to the shader. 
690          '''
691          time = Clock.get_boottime()
692          resolution = [float(size) for size in self.size]
693          self.canvas['time'] = time
694          self.canvas['resolution'] = resolution
695          for fbo in self.fbo_list:
696              fbo['time'] = time
697              fbo['resolution'] = resolution
698  
699      def refresh_fbo_setup(self, *args):
700          '''(internal) Creates and assigns one :class:`~kivy.graphics.Fbo` 
701          per effect, and makes sure all sizes etc. are correct and 
702          consistent. 
703          '''
704          # Add/remove fbos until there is one per effect
705          while len(self.fbo_list) < len(self.effects):
706              with self.canvas:
707                  new_fbo = EffectFbo(size=self.size)
708              with new_fbo:
709                  ClearColor(0, 0, 0, 0)
710                  ClearBuffers()
711                  Color(1, 1, 1, 1)
712                  new_fbo.texture_rectangle = Rectangle(size=self.size)
713  
714                  new_fbo.texture_rectangle.size = self.size
715              self.fbo_list.append(new_fbo)
716          while len(self.fbo_list) > len(self.effects):
717              old_fbo = self.fbo_list.pop()
718              self.canvas.remove(old_fbo)
719  
720          # Remove fbos from unused effects
721          for effect in self._bound_effects:
722              if effect not in self.effects:
723                  effect.fbo = None
724          self._bound_effects = self.effects
725  
726          # Do resizing etc.
727          self.fbo.size = self.size
728          self.fbo_rectangle.size = self.size
729          for i in range(len(self.fbo_list)):
730              self.fbo_list[i].size = self.size
731              self.fbo_list[i].texture_rectangle.size = self.size
732  
733          # If there are no effects, just draw our main fbo
734          if len(self.fbo_list) == 0:
735              self.texture = self.fbo.texture
736              return
737  
738          for i in range(1, len(self.fbo_list)):
739              fbo = self.fbo_list[i]
740              fbo.texture_rectangle.texture = self.fbo_list[i - 1].texture
741  
742          # Build effect shaders
743          for effect, fbo in zip(self.effects, self.fbo_list):
744              effect.fbo = fbo
745  
746          self.fbo_list[0].texture_rectangle.texture = self.fbo.texture
747          self.texture = self.fbo_list[-1].texture
748  
749          for fbo in self.fbo_list:
750              fbo.draw()
751          self.fbo.draw()
752  
753      def add_widget(self, widget):
754          # Add the widget to our Fbo instead of the normal canvas
755          c = self.canvas
756          self.canvas = self.fbo
757          super(EffectWidget, self).add_widget(widget)
758          self.canvas = c
759  
760      def remove_widget(self, widget):
761          # Remove the widget from our Fbo instead of the normal canvas
762          c = self.canvas
763          self.canvas = self.fbo
764          super(EffectWidget, self).remove_widget(widget)
765          self.canvas = c
766  
767      def clear_widgets(self, children=None):
768          # Clear widgets from our Fbo instead of the normal canvas
769          c = self.canvas
770          self.canvas = self.fbo
771          super(EffectWidget, self).clear_widgets(children)
772          self.canvas = c
