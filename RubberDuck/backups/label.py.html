1    '''Label 
2    ===== 
3     
4    .. image:: images/label.png 
5        :align: right 
6     
7    The :class:`Label` widget is for rendering text. It supports ascii and unicode 
8    strings:: 
9     
10       # hello world text 
11       l = Label(text='Hello world') 
12    
13       # unicode text; can only display glyphs that are available in the font 
14       l = Label(text=u'Hello world ' + unichr(2764)) 
15    
16       # multiline text 
17       l = Label(text='Multi\\nLine') 
18    
19       # size 
20       l = Label(text='Hello world', font_size='20sp') 
21    
22   .. _kivy-uix-label-sizing-and-text-content: 
23    
24   Sizing and text content 
25   --------------------------- 
26    
27   By default, the size of :class:`Label` is not affected by :attr:`~Label.text` 
28   content and the text is not affected by the size. In order to control 
29   sizing, you must specify :attr:`~Label.text_size` to constrain the text 
30   and/or bind :attr:`~Label.size` to :attr:`~Label.texture_size` to grow with 
31   the text. 
32    
33   For example, this label's size will be set to the text content 
34   (plus :attr:`~Label.padding`): 
35    
36   .. code-block:: kv 
37    
38       Label: 
39           size: self.texture_size 
40    
41   This label's text will wrap at the specified width and be clipped to the 
42   height: 
43    
44   .. code-block:: kv 
45    
46       Label: 
47           text_size: cm(6), cm(4) 
48    
49   .. note:: The :attr:`~Label.shorten` and :attr:`~Label.max_lines` attributes 
50    control how overflowing text behaves. 
51    
52   Combine these concepts to create a Label that can grow vertically but wraps the 
53   text at a certain width: 
54    
55   .. code-block:: kv 
56    
57       Label: 
58           text_size: root.width, None 
59           size: self.texture_size 
60    
61   How to have a custom background color in the label: 
62    
63   .. code-block:: kv 
64    
65       # Define your background color Template 
66       <BackgroundColor@Widget> 
67           background_color: 1, 1, 1, 1 
68           canvas.before: 
69               Color: 
70                   rgba: root.background_color 
71               Rectangle: 
72                   size: self.size 
73                   pos: self.pos 
74       # Now you can simply Mix the `BackgroundColor` class with almost 
75       # any other widget... to give it a background. 
76       <BackgroundLabel@Label+BackgroundColor> 
77           background_color: 0, 0, 0, 0 
78           # Default the background color for this label 
79           # to r 0, g 0, b 0, a 0 
80       # Use the BackgroundLabel any where in your kv code like below 
81       BackgroundLabel 
82           text: 'Hello' 
83           background_color: 1, 0, 0, 1 
84    
85   Text alignment and wrapping 
86   --------------------------- 
87    
88   The :class:`Label` has :attr:`~Label.halign` and :attr:`~Label.valign` 
89   properties to control the alignment of its text. However, by default the text 
90   image (:attr:`~Label.texture`) is only just large enough to contain the 
91   characters and is positioned in the center of the Label. The valign property 
92   will have no effect and halign will only have an effect if your text has 
93   newlines; a single line of text will appear to be centered even though halign 
94   is set to left (by default). 
95    
96   In order for the alignment properties to take effect, set the 
97   :attr:`~Label.text_size`, which specifies the size of the bounding box within 
98   which text is aligned. For instance, the following code binds this size to the 
99   size of the Label, so text will be aligned within the widget bounds. This 
100  will also automatically wrap the text of the Label to remain within this area. 
101   
102  .. code-block:: kv 
103   
104      Label: 
105          text_size: self.size 
106          halign: 'right' 
107          valign: 'middle' 
108   
109  Markup text 
110  ----------- 
111   
112  .. versionadded:: 1.1.0 
113   
114  You can change the style of the text using :doc:`api-kivy.core.text.markup`. 
115  The syntax is similar to the bbcode syntax but only the inline styling is 
116  allowed:: 
117   
118      # hello world with world in bold 
119      l = Label(text='Hello [b]World[/b]', markup=True) 
120   
121      # hello in red, world in blue 
122      l = Label(text='[color=ff3333]Hello[/color][color=3333ff]World[/color]', 
123          markup = True) 
124   
125  If you need to escape the markup from the current text, use 
126  :func:`kivy.utils.escape_markup`:: 
127   
128      text = 'This is an important message [1]' 
129      l = Label(text='[b]' + escape_markup(text) + '[/b]', markup=True) 
130   
131  The following tags are available: 
132   
133  ``[b][/b]`` 
134      Activate bold text 
135  ``[i][/i]`` 
136      Activate italic text 
137  ``[u][/u]`` 
138      Underlined text 
139  ``[s][/s]`` 
140      Strikethrough text 
141  ``[font=<str>][/font]`` 
142      Change the font (note: this refers to a TTF file or registered alias) 
143  ``[font_context=<str>][/font_context]`` 
144      Change context for the font, use string value "none" for isolated context 
145      (this is equivalent to `None`; if you created a font context named 
146      `'none'`, it cannot be referred to using markup) 
147  ``[font_family=<str>][/font_family]`` 
148      Font family to request for drawing. This is only valid when using a 
149      font context, see :class:`kivy.uix.label.Label` for details. 
150  ``[font_features=<str>][/font_features]`` 
151      OpenType font features, in CSS format, this is passed straight 
152      through to Pango. The effects of requesting a feature depends on loaded 
153      fonts, library versions, etc. Pango only, requires v1.38 or later. 
154  ``[size=<integer>][/size]`` 
155      Change the font size 
156  ``[color=#<color>][/color]`` 
157      Change the text color 
158  ``[ref=<str>][/ref]`` 
159      Add an interactive zone. The reference + bounding box inside the 
160      reference will be available in :attr:`Label.refs` 
161  ``[anchor=<str>]`` 
162      Put an anchor in the text. You can get the position of your anchor within 
163      the text with :attr:`Label.anchors` 
164  ``[sub][/sub]`` 
165      Display the text at a subscript position relative to the text before it. 
166  ``[sup][/sup]`` 
167      Display the text at a superscript position relative to the text before it. 
168  ``[text_language=<str>][/text_language]`` 
169      Language of the text, this is an RFC-3066 format language tag (as string), 
170      for example "en_US", "zh_CN", "fr" or "ja". This can impact font selection 
171      and metrics. Use the string "None" to revert to locale detection. 
172      Pango only. 
173   
174  If you want to render the markup text with a [ or ] or & character, you need to 
175  escape them. We created a simple syntax:: 
176   
177      [   -> &bl; 
178      ]   -> &br; 
179      &   -> &amp; 
180   
181  Then you can write:: 
182   
183      "[size=24]Hello &bl;World&br;[/size]" 
184   
185  Interactive zone in text 
186  ------------------------ 
187   
188  .. versionadded:: 1.1.0 
189   
190  You can now have definable "links" using text markup. The idea is to be able 
191  to detect when the user clicks on part of the text and to react. 
192  The tag ``[ref=xxx]`` is used for that. 
193   
194  In this example, we are creating a reference on the word "World". When 
195  this word is clicked, the function ``print_it`` will be called with the 
196  name of the reference:: 
197   
198      def print_it(instance, value): 
199          print('User clicked on', value) 
200      widget = Label(text='Hello [ref=world]World[/ref]', markup=True) 
201      widget.bind(on_ref_press=print_it) 
202   
203  For prettier rendering, you could add a color for the reference. Replace the 
204  ``text=`` in the previous example with:: 
205   
206      'Hello [ref=world][color=0000ff]World[/color][/ref]' 
207   
208  Catering for Unicode languages 
209  ------------------------------ 
210   
211  The font kivy uses does not contain all the characters required for displaying 
212  all languages. When you use the built-in widgets, this results in a block being 
213  drawn where you expect a character. 
214   
215  If you want to display such characters, you can chose a font that supports them 
216  and deploy it universally via kv: 
217   
218  .. code-block:: kv 
219   
220      <Label>: 
221          font_name: '/<path>/<to>/<font>' 
222   
223  Note that this needs to be done before your widgets are loaded as kv rules are 
224  only applied at load time. 
225   
226  Usage example 
227  ------------- 
228   
229  The following example marks the anchors and references contained in a label:: 
230   
231      from kivy.app import App 
232      from kivy.uix.label import Label 
233      from kivy.clock import Clock 
234      from kivy.graphics import Color, Rectangle 
235   
236   
237      class TestApp(App): 
238   
239          @staticmethod 
240          def get_x(label, ref_x): 
241              """ Return the x value of the ref/anchor relative to the canvas """ 
242              return label.center_x - label.texture_size[0] * 0.5 + ref_x 
243   
244          @staticmethod 
245          def get_y(label, ref_y): 
246              """ Return the y value of the ref/anchor relative to the canvas """ 
247              # Note the inversion of direction, as y values start at the top of 
248              # the texture and increase downwards 
249              return label.center_y + label.texture_size[1] * 0.5 - ref_y 
250   
251          def show_marks(self, label): 
252   
253              # Indicate the position of the anchors with a red top marker 
254              for name, anc in label.anchors.items(): 
255                  with label.canvas: 
256                      Color(1, 0, 0) 
257                      Rectangle(pos=(self.get_x(label, anc[0]), 
258                                     self.get_y(label, anc[1])), 
259                                size=(3, 3)) 
260   
261              # Draw a green surround around the refs. Note the sizes y inversion 
262              for name, boxes in label.refs.items(): 
263                  for box in boxes: 
264                      with label.canvas: 
265                          Color(0, 1, 0, 0.25) 
266                          Rectangle(pos=(self.get_x(label, box[0]), 
267                                         self.get_y(label, box[1])), 
268                                    size=(box[2] - box[0], 
269                                          box[1] - box[3])) 
270   
271          def build(self): 
272              label = Label( 
273                  text='[anchor=a]a\\nChars [anchor=b]b\\n[ref=myref]ref[/ref]', 
274                  markup=True) 
275              Clock.schedule_once(lambda dt: self.show_marks(label), 1) 
276              return label 
277   
278      TestApp().run()

316      def __init__(self, **kwargs):
317          self._trigger_texture = Clock.create_trigger(self.texture_update, -1)
318          super(Label, self).__init__(**kwargs)
319  
320          # bind all the property for recreating the texture
321          d = Label._font_properties
322          fbind = self.fbind
323          update = self._trigger_texture_update
324          fbind('disabled', update, 'disabled')
325          for x in d:
326              fbind(x, update, x)
327  
328          self._label = None
329          self._create_label()
330  
331          # force the texture creation
332          self._trigger_texture()
333  
334      def _create_label(self):
335          # create the core label class according to markup value
336          if self._label is not None:
337              cls = self._label.__class__
338          else:
339              cls = None
340          markup = self.markup
341          if (markup and cls is not CoreMarkupLabel) or \
342             (not markup and cls is not CoreLabel):
343              # markup have change, we need to change our rendering method.
344              d = Label._font_properties
345              dkw = dict(list(zip(d, [getattr(self, x) for x in d])))
346              if markup:
347                  self._label = CoreMarkupLabel(**dkw)
348              else:
349                  self._label = CoreLabel(**dkw)
350  
351      def _trigger_texture_update(self, name=None, source=None, value=None):
352          # check if the label core class need to be switch to a new one
353          if name == 'markup':
354              self._create_label()
355          if source:
356              if name == 'text':
357                  self._label.text = value
358              elif name == 'text_size':
359                  self._label.usersize = value
360              elif name == 'font_size':
361                  self._label.options[name] = value
362              elif name == 'disabled_color' and self.disabled:
363                  self._label.options['color'] = value
364              elif name == 'disabled_outline_color' and self.disabled:
365                  self._label.options['outline_color'] = value
366              elif name == 'disabled':
367                  self._label.options['color'] = self.disabled_color if value \
368                      else self.color
369                  self._label.options['outline_color'] = (
370                      self.disabled_outline_color if value else
371                      self.outline_color)
372              else:
373                  self._label.options[name] = value
374          self._trigger_texture()
375  
376      def texture_update(self, *largs):
377          '''Force texture recreation with the current Label properties. 
378   
379          After this function call, the :attr:`texture` and :attr:`texture_size` 
380          will be updated in this order. 
381          '''
382          mrkup = self._label.__class__ is CoreMarkupLabel
383          self.texture = None
384  
385          if (not self._label.text or
386                  (self.halign == 'justify' or self.strip) and
387                  not self._label.text.strip()):
388              self.texture_size = (0, 0)
389              self.is_shortened = False
390              if mrkup:
391                  self.refs, self._label._refs = {}, {}
392                  self.anchors, self._label._anchors = {}, {}
393          else:
394              if mrkup:
395                  text = self.text
396                  # we must strip here, otherwise, if the last line is empty,
397                  # markup will retain the last empty line since it only strips
398                  # line by line within markup
399                  if self.halign == 'justify' or self.strip:
400                      text = text.strip()
401                  self._label.text = ''.join(('[color=',
402                                              get_hex_from_color(
403                                                  self.disabled_color if
404                                                  self.disabled else self.color),
405                                              ']', text, '[/color]'))
406                  self._label.refresh()
407                  # force the rendering to get the references
408                  if self._label.texture:
409                      self._label.texture.bind()
410                  self.refs = self._label.refs
411                  self.anchors = self._label.anchors
412              else:
413                  self._label.refresh()
414              texture = self._label.texture
415              if texture is not None:
416                  self.texture = self._label.texture
417                  self.texture_size = list(self.texture.size)
418              self.is_shortened = self._label.is_shortened
419  
420      def on_touch_down(self, touch):
421          if super(Label, self).on_touch_down(touch):
422              return True
423          if not len(self.refs):
424              return False
425          tx, ty = touch.pos
426          tx -= self.center_x - self.texture_size[0] / 2.
427          ty -= self.center_y - self.texture_size[1] / 2.
428          ty = self.texture_size[1] - ty
429          for uid, zones in self.refs.items():
430              for zone in zones:
431                  x, y, w, h = zone
432                  if x <= tx <= w and y <= ty <= h:
433                      self.dispatch('on_ref_press', uid)
434                      return True
435          return False
436  
437      def on_ref_press(self, ref):
438          pass
