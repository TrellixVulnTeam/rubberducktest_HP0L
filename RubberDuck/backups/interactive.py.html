1    ''' 
2    Interactive launcher 
3    ==================== 
4     
5    .. versionadded:: 1.3.0 
6     
7    .. deprecated:: 1.10.0 
8        The interactive launcher has been deprecated. 
9     
10   The :class:`InteractiveLauncher` provides a user-friendly python shell 
11   interface to an :class:`App` so that it can be prototyped and debugged 
12   interactively. 
13    
14   .. note:: 
15    
16       The Kivy API intends for some functions to only be run once or before the 
17       main EventLoop has started. Methods that can normally be called during the 
18       course of an application will work as intended, but specifically overriding 
19       methods such as :meth:`on_touch` dynamically leads to trouble. 
20    
21   Creating an InteractiveLauncher 
22   ------------------------------- 
23    
24   Take your existing subclass of :class:`App` (this can be production code) and 
25   pass an instance to the :class:`InteractiveLauncher` constructor. :: 
26    
27       from kivy.interactive import InteractiveLauncher 
28       from kivy.app import App 
29       from kivy.uix.button import Button 
30    
31       class MyApp(App): 
32           def build(self): 
33               return Button(text='Hello Shell') 
34    
35       launcher = InteractiveLauncher(MyApp()) 
36       launcher.run() 
37    
38   After pressing *enter*, the script will return. This allows the interpreter to 
39   continue running. Inspection or modification of the :class:`App` can be done 
40   safely through the InteractiveLauncher instance or the provided 
41   :class:`SafeMembrane` class instances. 
42    
43   .. note:: 
44    
45       If you want to test this example, start Python without any file to have 
46       already an interpreter, and copy/paste all the lines. You'll still have the 
47       interpreter at the end + the kivy application running. 
48    
49   Interactive Development 
50   ----------------------- 
51    
52   IPython provides a fast way to learn the Kivy API. The :class:`App` instance 
53   and all of its attributes, including methods and the entire widget tree, 
54   can be quickly listed by using the '.' operator and pressing 'tab'. Try this 
55   code in an Ipython shell. :: 
56    
57       from kivy.interactive import InteractiveLauncher 
58       from kivy.app import App 
59       from kivy.uix.widget import Widget 
60       from kivy.graphics import Color, Ellipse 
61    
62       class MyPaintWidget(Widget): 
63           def on_touch_down(self, touch): 
64               with self.canvas: 
65                   Color(1, 1, 0) 
66                   d = 30. 
67                   Ellipse(pos=(touch.x - d/2, touch.y - d/2), size=(d, d)) 
68    
69    
70       class TestApp(App): 
71           def build(self): 
72               return Widget() 
73    
74    
75       i = InteractiveLauncher(TestApp()) 
76       i.run() 
77       i.       # press 'tab' to list attributes of the app 
78       i.root.  # press 'tab' to list attributes of the root widget 
79    
80       # App is boring. Attach a new widget! 
81       i.root.add_widget(MyPaintWidget()) 
82    
83       i.safeIn() 
84       # The application is now blocked. 
85       # Click on the screen several times. 
86       i.safeOut() 
87       # The clicks will show up now 
88    
89       # Erase artwork and start over 
90       i.root.canvas.clear() 
91    
92   .. note:: 
93    
94       All of the proxies used in the module store their referent in the 
95       :attr:`_ref` attribute, which can be accessed directly if needed, such as 
96       for getting doc strings. :func:`help` and :func:`type` will access the 
97       proxy, not its referent. 
98    
99   Directly Pausing the Application 
100  -------------------------------- 
101   
102  Both the :class:`InteractiveLauncher` and :class:`SafeMembrane` hold internal 
103  references to the :class:`EventLoop`'s 'safe' and 'confirmed' 
104  :class:`threading.Event` objects. You can use their safing methods to control 
105  the application manually. 
106   
107  :meth:`SafeMembrane.safeIn` will cause the application to pause and 
108  :meth:`SafeMembrane.safeOut` will allow a paused application 
109  to continue running. This is potentially useful for scripting actions into 
110  functions that need the screen to update etc. 
111   
112  .. note:: 
113   
114      The pausing is implemented via the 
115      :class:`Clocks' <kivy.clock.Clock>` 
116      :meth:`~kivy.clock.ClockBase.schedule_once` method 
117      and occurs before the start of each frame. 
118   
119  Adding Attributes Dynamically 
120  ----------------------------- 
121   
122  .. note:: 
123   
124      This module uses threading and object proxies to encapsulate the running 
125      :class:`App`. Deadlocks and memory corruption can occur if making direct 
126      references inside the thread without going through the provided proxy(s). 
127   
128  The :class:`InteractiveLauncher` can have attributes added to it exactly like a 
129  normal object and if these were created from outside the membrane, they will 
130  not be threadsafe because the external references to them in the python 
131  interpreter do not go through InteractiveLauncher's membrane behavior, 
132  inherited from :class:`SafeMembrane`. 
133   
134  To threadsafe these external references, simply assign them to 
135  :class:`SafeMembrane` instances of themselves like so:: 
136   
137      from kivy.interactive import SafeMembrane 
138   
139      interactiveLauncher.attribute = myNewObject 
140      # myNewObject is unsafe 
141      myNewObject = SafeMembrane(myNewObject) 
142      # myNewObject is now safe. Call at will. 
143      myNewObject.method() 
144

172      EventLoop.confirmed.set()
173      EventLoop.safe.wait()

177  def unwrap(ob):
178      while type(ob) == SafeMembrane:
179          ob = ob._ref
180      return ob

183  class SafeMembrane(object):
184      ''' 
185      This help is for a proxy object. Did you want help on the proxy's referent 
186      instead? Try using help(<instance>._ref) 
187   
188      The SafeMembrane is a threadsafe proxy that also returns attributes as new 
189      thread-safe objects 
190      and makes thread-safe method calls, preventing thread-unsafe objects 
191      from leaking into the user's environment. 
192      '''
193  
194      __slots__ = ('_ref', 'safe', 'confirmed')
195  
196      def __init__(self, ob, *args, **kwargs):
197          self.confirmed = EventLoop.confirmed
198          self.safe = EventLoop.safe
199          self._ref = ob
200  
201      def safeIn(self):
202          """Provides a thread-safe entry point for interactive launching."""
203          self.safe.clear()
204          Clock.schedule_once(safeWait, -1)
205          self.confirmed.wait()
206  
207      def safeOut(self):
208          """Provides a thread-safe exit point for interactive launching."""
209          self.safe.set()
210  
211      def isMethod(self, fn):
212          return inspect.ismethod(fn)
213  
214      # Everything from this point on is just a series of thread-safing proxy
215      # methods that make calls against _ref and threadsafe whenever data will be
216      # written to or if a method will be called. SafeMembrane instances should
217      # be unwrapped whenever passing them into the thread
218      # use type() to determine if an object is a SafeMembrane while debugging
219      def __repr__(self):
220          return self._ref.__repr__()
221  
222      def __call__(self, *args, **kw):
223          self.safeIn()
224          args = list(map(unwrap, args))
225          for k in list(kw.keys()):
226              kw[k] = unwrap(kw[k])
227          r = self._ref(*args, **kw)
228          self.safeOut()
229          if r is not None:
230              return SafeMembrane(r)
231  
232      def __getattribute__(self, attr, oga=object.__getattribute__):
233          if attr.startswith('__') or attr == '_ref':
234              subject = oga(self, '_ref')
235              if attr == '_ref':
236                  return subject
237              return getattr(subject, attr)
238          return oga(self, attr)
239  
240      def __getattr__(self, attr, oga=object.__getattribute__):
241          r = getattr(oga(self, '_ref'), attr)
242          return SafeMembrane(r)
243  
244      def __setattr__(self, attr, val, osa=object.__setattr__):
245          if (attr == '_ref' or
246                  hasattr(type(self), attr) and not attr.startswith('__')):
247              osa(self, attr, val)
248          else:
249              self.safeIn()
250              val = unwrap(val)
251              setattr(self._ref, attr, val)
252              self.safeOut()
253  
254      def __delattr__(self, attr, oda=object.__delattr__):
255          self.safeIn()
256          delattr(self._ref, attr)
257          self.safeOut()
258  
259      def __bool__(self):
260          return bool(self._ref)
261  
262      def __getitem__(self, arg):
263          return SafeMembrane(self._ref[arg])
264  
265      def __setitem__(self, arg, val):
266          self.safeIn()
267          val = unwrap(val)
268          self._ref[arg] = val
269          self.safeOut()
270  
271      def __delitem__(self, arg):
272          self.safeIn()
273          del self._ref[arg]
274          self.safeOut()
275  
276      def __getslice__(self, i, j):
277          return SafeMembrane(self._ref[i:j])
278  
279      def __setslice__(self, i, j, val):
280          self.safeIn()
281          val = unwrap(val)
282          self._ref[i:j] = val
283          self.safeOut()
284  
285      def __delslice__(self, i, j):
286          self.safeIn()
287          del self._ref[i:j]
288          self.safeOut()
289  
290      def __enter__(self, *args, **kwargs):
291          self.safeIn()
292          self._ref.__enter__(*args, **kwargs)
293  
294      def __exit__(self, *args, **kwargs):
295          self._ref.__exit__(*args, **kwargs)
296          self.safeOut()

299  class InteractiveLauncher(SafeMembrane):
300      ''' 
301      Proxy to an application instance that launches it in a thread and 
302      then returns and acts as a proxy to the application in the thread. 
303      '''
304  
305      __slots__ = ('_ref', 'safe', 'confirmed', 'thread', 'app')
306  
307      @deprecated
308      def __init__(self, app=None, *args, **kwargs):
309          if app is None:
310              app = App()
311          EventLoop.safe = Event()
312          self.safe = EventLoop.safe
313          self.safe.set()
314          EventLoop.confirmed = Event()
315          self.confirmed = EventLoop.confirmed
316          self.app = app
317  
318          def startApp(app=app, *args, **kwargs):
319              app.run(*args, **kwargs)
320  
321          self.thread = Thread(target=startApp, *args, **kwargs)
322  
323      def run(self):
324          self.thread.start()
325          # Proxy behavior starts after this is set. Before this point, attaching
326          # widgets etc can only be done through the Launcher's app attribute
327          self._ref = self.app
328  
329      def stop(self):
330          EventLoop.quit = True
331          self.thread.join()
332  
333      # Act like the app instance even before _ref is set
334      def __repr__(self):
335          return self.app.__repr__()
