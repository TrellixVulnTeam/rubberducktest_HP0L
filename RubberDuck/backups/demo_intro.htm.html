16   <H3>Object model</H3>
17   <P>Except as described below, the object module exposed should be similar to that exposed
18   by Visual Basic, etc.  Due to the nature of ActiveX Scripting, the details for each
19   host are different, but Python should work "correctly".
20   
21   <P>The object model exposed via Python for MSIE is not as seamless as VB.  The biggest limitation is 
22   the concept of a "local" namespace.  For example, in VB, you can
23   code <code>text="Hi there"</code>, but in Python, you must code 
24   <code>MyForm.ThisButton.Text="Hi There"</code>.  See the <A HREF="foo2.htm">foo2</A> sample 
25   for futher details.
