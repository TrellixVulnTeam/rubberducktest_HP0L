1    """Policies  
2     
3    Note that Dispatchers are now implemented in "dispatcher.py", but 
4    are still documented here. 
5     
6    Policies 
7     
8     A policy is an object which manages the interaction between a public  
9     Python object, and COM .  In simple terms, the policy object is the  
10    object which is actually called by COM, and it invokes the requested  
11    method, fetches/sets the requested property, etc.  See the  
12    @win32com.server.policy.CreateInstance@ method for a description of 
13    how a policy is specified or created. 
14    
15    Exactly how a policy determines which underlying object method/property  
16    is obtained is up to the policy.  A few policies are provided, but you  
17    can build your own.  See each policy class for a description of how it  
18    implements its policy. 
19    
20    There is a policy that allows the object to specify exactly which  
21    methods and properties will be exposed.  There is also a policy that  
22    will dynamically expose all Python methods and properties - even those  
23    added after the object has been instantiated. 
24    
25   Dispatchers 
26    
27    A Dispatcher is a level in front of a Policy.  A dispatcher is the  
28    thing which actually receives the COM calls, and passes them to the  
29    policy object (which in turn somehow does something with the wrapped  
30    object). 
31    
32    It is important to note that a policy does not need to have a dispatcher. 
33    A dispatcher has the same interface as a policy, and simply steps in its  
34    place, delegating to the real policy.  The primary use for a Dispatcher  
35    is to support debugging when necessary, but without imposing overheads  
36    when not (ie, by not using a dispatcher at all). 
37    
38    There are a few dispatchers provided - "tracing" dispatchers which simply  
39    prints calls and args (including a variation which uses  
40    win32api.OutputDebugString), and a "debugger" dispatcher, which can  
41    invoke the debugger when necessary. 
42    
43   Error Handling 
44    
45    It is important to realise that the caller of these interfaces may 
46    not be Python.  Therefore, general Python exceptions and tracebacks aren't  
47    much use. 
48    
49    In general, there is an Exception class that should be raised, to allow  
50    the framework to extract rich COM type error information. 
51    
52    The general rule is that the **only** exception returned from Python COM  
53    Server code should be an Exception instance.  Any other Python exception  
54    should be considered an implementation bug in the server (if not, it  
55    should be handled, and an appropriate Exception instance raised).  Any  
56    other exception is considered "unexpected", and a dispatcher may take  
57    special action (see Dispatchers above) 
58    
59    Occasionally, the implementation will raise the policy.error error.   
60    This usually means there is a problem in the implementation that the  
61    Python programmer should fix. 
62    
63    For example, if policy is asked to wrap an object which it can not  
64    support (because, eg, it does not provide _public_methods_ or _dynamic_)  
65    then policy.error will be raised, indicating it is a Python programmers  
66    problem, rather than a COM error. 
67

111  def CreateInstance(clsid, reqIID):
112      """Create a new instance of the specified IID 
113   
114      The COM framework **always** calls this function to create a new 
115      instance for the specified CLSID.  This function looks up the 
116      registry for the name of a policy, creates the policy, and asks the 
117      policy to create the specified object by calling the _CreateInstance_ method. 
118   
119      Exactly how the policy creates the instance is up to the policy.  See the 
120      specific policy documentation for more details. 
121      """
122      # First see is sys.path should have something on it.
123      try:
124          addnPaths = win32api.RegQueryValue(
125              win32con.HKEY_CLASSES_ROOT, regAddnPath % clsid
126          ).split(";")
127          for newPath in addnPaths:
128              if newPath not in sys.path:
129                  sys.path.insert(0, newPath)
130      except win32api.error:
131          pass
132      try:
133          policy = win32api.RegQueryValue(win32con.HKEY_CLASSES_ROOT, regPolicy % clsid)
134          policy = resolve_func(policy)
135      except win32api.error:
136          policy = DefaultPolicy
137  
138      try:
139          dispatcher = win32api.RegQueryValue(
140              win32con.HKEY_CLASSES_ROOT, regDispatcher % clsid
141          )
142          if dispatcher:
143              dispatcher = resolve_func(dispatcher)
144      except win32api.error:
145          dispatcher = None
146  
147      if dispatcher:
148          retObj = dispatcher(policy, None)
149      else:
150          retObj = policy(None)
151      return retObj._CreateInstance_(clsid, reqIID)

154  class BasicWrapPolicy:
155      """The base class of policies. 
156   
157      Normally not used directly (use a child class, instead) 
158   
159      This policy assumes we are wrapping another object 
160      as the COM server.  This supports the delegation of the core COM entry points 
161      to either the wrapped object, or to a child class. 
162   
163      This policy supports the following special attributes on the wrapped object 
164   
165      _query_interface_ -- A handler which can respond to the COM 'QueryInterface' call. 
166      _com_interfaces_ -- An optional list of IIDs which the interface will assume are 
167          valid for the object. 
168      _invoke_ -- A handler which can respond to the COM 'Invoke' call.  If this attribute 
169          is not provided, then the default policy implementation is used.  If this attribute 
170          does exist, it is responsible for providing all required functionality - ie, the 
171          policy _invoke_ method is not invoked at all (and nor are you able to call it!) 
172      _getidsofnames_ -- A handler which can respond to the COM 'GetIDsOfNames' call.  If this attribute 
173          is not provided, then the default policy implementation is used.  If this attribute 
174          does exist, it is responsible for providing all required functionality - ie, the 
175          policy _getidsofnames_ method is not invoked at all (and nor are you able to call it!) 
176   
177      IDispatchEx functionality: 
178   
179      _invokeex_ -- Very similar to _invoke_, except slightly different arguments are used. 
180          And the result is just the _real_ result (rather than the (hresult, argErr, realResult) 
181          tuple that _invoke_ uses. 
182          This is the new, prefered handler (the default _invoke_ handler simply called _invokeex_) 
183      _getdispid_ -- Very similar to _getidsofnames_, except slightly different arguments are used, 
184          and only 1 property at a time can be fetched (which is all we support in getidsofnames anyway!) 
185          This is the new, prefered handler (the default _invoke_ handler simply called _invokeex_) 
186      _getnextdispid_- uses self._name_to_dispid_ to enumerate the DISPIDs 
187      """
188  
189      def __init__(self, object):
190          """Initialise the policy object 
191   
192          Params: 
193   
194          object -- The object to wrap.  May be None *iff* @BasicWrapPolicy._CreateInstance_@ will be 
195          called immediately after this to setup a brand new object 
196          """
197          if object is not None:
198              self._wrap_(object)
199  
200      def _CreateInstance_(self, clsid, reqIID):
201          """Creates a new instance of a **wrapped** object 
202   
203          This method looks up a "@win32com.server.policy.regSpec@" % clsid entry 
204          in the registry (using @DefaultPolicy@) 
205          """
206          try:
207              classSpec = win32api.RegQueryValue(
208                  win32con.HKEY_CLASSES_ROOT, regSpec % clsid
209              )
210          except win32api.error:
211              raise error(
212                  "The object is not correctly registered - %s key can not be read"
213                  % (regSpec % clsid)
214              )
215          myob = call_func(classSpec)
216          self._wrap_(myob)
217          try:
218              return pythoncom.WrapObject(self, reqIID)
219          except pythoncom.com_error as xxx_todo_changeme:
220              (hr, desc, exc, arg) = xxx_todo_changeme.args
221              from win32com.util import IIDToInterfaceName
222  
223              desc = (
224                  "The object '%r' was created, but does not support the "
225                  "interface '%s'(%s): %s"
226                  % (myob, IIDToInterfaceName(reqIID), reqIID, desc)
227              )
228              raise pythoncom.com_error(hr, desc, exc, arg)

230      def _wrap_(self, object):
231          """Wraps up the specified object. 
232   
233          This function keeps a reference to the passed 
234          object, and may interogate it to determine how to respond to COM requests, etc. 
235          """
236          # We "clobber" certain of our own methods with ones
237          # provided by the wrapped object, iff they exist.
238          self._name_to_dispid_ = {}
239          ob = self._obj_ = object
240          if hasattr(ob, "_query_interface_"):
241              self._query_interface_ = ob._query_interface_
242  
243          if hasattr(ob, "_invoke_"):
244              self._invoke_ = ob._invoke_
245  
246          if hasattr(ob, "_invokeex_"):
247              self._invokeex_ = ob._invokeex_
248  
249          if hasattr(ob, "_getidsofnames_"):
250              self._getidsofnames_ = ob._getidsofnames_
251  
252          if hasattr(ob, "_getdispid_"):
253              self._getdispid_ = ob._getdispid_
254  
255          # Allow for override of certain special attributes.
256          if hasattr(ob, "_com_interfaces_"):
257              self._com_interfaces_ = []
258              # Allow interfaces to be specified by name.
259              for i in ob._com_interfaces_:
260                  if type(i) != pywintypes.IIDType:
261                      # Prolly a string!
262                      if i[0] != "{":
263                          i = pythoncom.InterfaceNames[i]
264                      else:
265                          i = pythoncom.MakeIID(i)
266                  self._com_interfaces_.append(i)
267          else:
268              self._com_interfaces_ = []
269  
270      # "QueryInterface" handling.
271      def _QueryInterface_(self, iid):
272          """The main COM entry-point for QueryInterface. 
273   
274          This checks the _com_interfaces_ attribute and if the interface is not specified 
275          there, it calls the derived helper _query_interface_ 
276          """
277          if iid in self._com_interfaces_:
278              return 1
279          return self._query_interface_(iid)
280  
281      def _query_interface_(self, iid):
282          """Called if the object does not provide the requested interface in _com_interfaces_, 
283          and does not provide a _query_interface_ handler. 
284   
285          Returns a result to the COM framework indicating the interface is not supported. 
286          """
287          return 0

289      # "Invoke" handling.
290      def _Invoke_(self, dispid, lcid, wFlags, args):
291          """The main COM entry-point for Invoke. 
292   
293          This calls the _invoke_ helper. 
294          """
295          # Translate a possible string dispid to real dispid.
296          if type(dispid) == type(""):
297              try:
298                  dispid = self._name_to_dispid_[dispid.lower()]
299              except KeyError:
300                  raise COMException(
301                      scode=winerror.DISP_E_MEMBERNOTFOUND, desc="Member not found"
302                  )
303          return self._invoke_(dispid, lcid, wFlags, args)
304  
305      def _invoke_(self, dispid, lcid, wFlags, args):
306          # Delegates to the _invokeex_ implementation.  This allows
307          # a custom policy to define _invokeex_, and automatically get _invoke_ too.
308          return S_OK, -1, self._invokeex_(dispid, lcid, wFlags, args, None, None)

310      # "GetIDsOfNames" handling.
311      def _GetIDsOfNames_(self, names, lcid):
312          """The main COM entry-point for GetIDsOfNames. 
313   
314          This checks the validity of the arguments, and calls the _getidsofnames_ helper. 
315          """
316          if len(names) > 1:
317              raise COMException(
318                  scode=winerror.DISP_E_INVALID,
319                  desc="Cannot support member argument names",
320              )
321          return self._getidsofnames_(names, lcid)

323      def _getidsofnames_(self, names, lcid):
324          ### note: lcid is being ignored...
325          return (self._getdispid_(names[0], 0),)

392      def _getnextdispid_(self, fdex, dispid):
393          ids = list(self._name_to_dispid_.values())
394          ids.sort()
395          if DISPID_STARTENUM in ids:
396              ids.remove(DISPID_STARTENUM)
397          if dispid == DISPID_STARTENUM:
398              return ids[0]
399          else:
400              try:
401                  return ids[ids.index(dispid) + 1]
402              except ValueError:  # dispid not in list?
403                  raise COMException(scode=winerror.E_UNEXPECTED)
404              except IndexError:  # No more items
405                  raise COMException(scode=winerror.S_FALSE)

414  class MappedWrapPolicy(BasicWrapPolicy):
415      """Wraps an object using maps to do its magic 
416   
417      This policy wraps up a Python object, using a number of maps 
418      which translate from a Dispatch ID and flags, into an object to call/getattr, etc. 
419   
420      It is the responsibility of derived classes to determine exactly how the 
421      maps are filled (ie, the derived classes determine the map filling policy. 
422   
423      This policy supports the following special attributes on the wrapped object 
424   
425      _dispid_to_func_/_dispid_to_get_/_dispid_to_put_ -- These are dictionaries 
426        (keyed by integer dispid, values are string attribute names) which the COM 
427        implementation uses when it is processing COM requests.  Note that the implementation 
428        uses this dictionary for its own purposes - not a copy - which means the contents of 
429        these dictionaries will change as the object is used. 
430   
431      """
432  
433      def _wrap_(self, object):
434          BasicWrapPolicy._wrap_(self, object)
435          ob = self._obj_
436          if hasattr(ob, "_dispid_to_func_"):
437              self._dispid_to_func_ = ob._dispid_to_func_
438          else:
439              self._dispid_to_func_ = {}
440          if hasattr(ob, "_dispid_to_get_"):
441              self._dispid_to_get_ = ob._dispid_to_get_
442          else:
443              self._dispid_to_get_ = {}
444          if hasattr(ob, "_dispid_to_put_"):
445              self._dispid_to_put_ = ob._dispid_to_put_
446          else:
447              self._dispid_to_put_ = {}
448  
449      def _getmembername_(self, dispid):
450          if dispid in self._dispid_to_func_:
451              return self._dispid_to_func_[dispid]
452          elif dispid in self._dispid_to_get_:
453              return self._dispid_to_get_[dispid]
454          elif dispid in self._dispid_to_put_:
455              return self._dispid_to_put_[dispid]
456          else:
457              raise COMException(scode=winerror.DISP_E_MEMBERNOTFOUND)

460  class DesignatedWrapPolicy(MappedWrapPolicy):
461      """A policy which uses a mapping to link functions and dispid 
462   
463       A MappedWrappedPolicy which allows the wrapped object to specify, via certain 
464       special named attributes, exactly which methods and properties are exposed. 
465   
466       All a wrapped object need do is provide the special attributes, and the policy 
467       will handle everything else. 
468   
469       Attributes: 
470   
471       _public_methods_ -- Required, unless a typelib GUID is given -- A list 
472                    of strings, which must be the names of methods the object 
473                    provides.  These methods will be exposed and callable 
474                    from other COM hosts. 
475       _public_attrs_ A list of strings, which must be the names of attributes on the object. 
476                    These attributes will be exposed and readable and possibly writeable from other COM hosts. 
477       _readonly_attrs_ -- A list of strings, which must also appear in _public_attrs.  These 
478                    attributes will be readable, but not writable, by other COM hosts. 
479       _value_ -- A method that will be called if the COM host requests the "default" method 
480                    (ie, calls Invoke with dispid==DISPID_VALUE) 
481       _NewEnum -- A method that will be called if the COM host requests an enumerator on the 
482                    object (ie, calls Invoke with dispid==DISPID_NEWENUM.) 
483                    It is the responsibility of the method to ensure the returned 
484                    object conforms to the required Enum interface. 
485   
486      _typelib_guid_ -- The GUID of the typelibrary with interface definitions we use. 
487      _typelib_version_ -- A tuple of (major, minor) with a default of 1,1 
488      _typelib_lcid_ -- The LCID of the typelib, default = LOCALE_USER_DEFAULT 
489   
490       _Evaluate -- Dunno what this means, except the host has called Invoke with dispid==DISPID_EVALUATE! 
491                    See the COM documentation for details. 
492      """
493  
494      def _wrap_(self, ob):
495          # If we have nominated universal interfaces to support, load them now
496          tlb_guid = getattr(ob, "_typelib_guid_", None)
497          if tlb_guid is not None:
498              tlb_major, tlb_minor = getattr(ob, "_typelib_version_", (1, 0))
499              tlb_lcid = getattr(ob, "_typelib_lcid_", 0)
500              from win32com import universal
501  
502              # XXX - what if the user wants to implement interfaces from multiple
503              # typelibs?
504              # Filter out all 'normal' IIDs (ie, IID objects and strings starting with {
505              interfaces = [
506                  i
507                  for i in getattr(ob, "_com_interfaces_", [])
508                  if type(i) != pywintypes.IIDType and not i.startswith("{")
509              ]
510              universal_data = universal.RegisterInterfaces(
511                  tlb_guid, tlb_lcid, tlb_major, tlb_minor, interfaces
512              )
513          else:
514              universal_data = []
515          MappedWrapPolicy._wrap_(self, ob)
516          if not hasattr(ob, "_public_methods_") and not hasattr(ob, "_typelib_guid_"):
517              raise error(
518                  "Object does not support DesignatedWrapPolicy, as it does not have either _public_methods_ or _typelib_guid_ attributes."
519              )
520  
521          # Copy existing _dispid_to_func_ entries to _name_to_dispid_
522          for dispid, name in self._dispid_to_func_.items():
523              self._name_to_dispid_[name.lower()] = dispid
524          for dispid, name in self._dispid_to_get_.items():
525              self._name_to_dispid_[name.lower()] = dispid
526          for dispid, name in self._dispid_to_put_.items():
527              self._name_to_dispid_[name.lower()] = dispid
528  
529          # Patch up the universal stuff.
530          for dispid, invkind, name in universal_data:
531              self._name_to_dispid_[name.lower()] = dispid
532              if invkind == DISPATCH_METHOD:
533                  self._dispid_to_func_[dispid] = name
534              elif invkind in (DISPATCH_PROPERTYPUT, DISPATCH_PROPERTYPUTREF):
535                  self._dispid_to_put_[dispid] = name
536              elif invkind == DISPATCH_PROPERTYGET:
537                  self._dispid_to_get_[dispid] = name
538              else:
539                  raise ValueError("unexpected invkind: %d (%s)" % (invkind, name))
540  
541          # look for reserved methods
542          if hasattr(ob, "_value_"):
543              self._dispid_to_get_[DISPID_VALUE] = "_value_"
544              self._dispid_to_put_[DISPID_PROPERTYPUT] = "_value_"
545          if hasattr(ob, "_NewEnum"):
546              self._name_to_dispid_["_newenum"] = DISPID_NEWENUM
547              self._dispid_to_func_[DISPID_NEWENUM] = "_NewEnum"
548          if hasattr(ob, "_Evaluate"):
549              self._name_to_dispid_["_evaluate"] = DISPID_EVALUATE
550              self._dispid_to_func_[DISPID_EVALUATE] = "_Evaluate"
551  
552          next_dispid = self._allocnextdispid(999)
553          # note: funcs have precedence over attrs (install attrs first)
554          if hasattr(ob, "_public_attrs_"):
555              if hasattr(ob, "_readonly_attrs_"):
556                  readonly = ob._readonly_attrs_
557              else:
558                  readonly = []
559              for name in ob._public_attrs_:
560                  dispid = self._name_to_dispid_.get(name.lower())
561                  if dispid is None:
562                      dispid = next_dispid
563                      self._name_to_dispid_[name.lower()] = dispid
564                      next_dispid = self._allocnextdispid(next_dispid)
565                  self._dispid_to_get_[dispid] = name
566                  if name not in readonly:
567                      self._dispid_to_put_[dispid] = name
568          for name in getattr(ob, "_public_methods_", []):
569              dispid = self._name_to_dispid_.get(name.lower())
570              if dispid is None:
571                  dispid = next_dispid
572                  self._name_to_dispid_[name.lower()] = dispid
573                  next_dispid = self._allocnextdispid(next_dispid)
574              self._dispid_to_func_[dispid] = name
575          self._typeinfos_ = None  # load these on demand.
576  
577      def _build_typeinfos_(self):
578          # Can only ever be one for now.
579          tlb_guid = getattr(self._obj_, "_typelib_guid_", None)
580          if tlb_guid is None:
581              return []
582          tlb_major, tlb_minor = getattr(self._obj_, "_typelib_version_", (1, 0))
583          tlb = pythoncom.LoadRegTypeLib(tlb_guid, tlb_major, tlb_minor)
584          typecomp = tlb.GetTypeComp()
585          # Not 100% sure what semantics we should use for the default interface.
586          # Look for the first name in _com_interfaces_ that exists in the typelib.
587          for iname in self._obj_._com_interfaces_:
588              try:
589                  type_info, type_comp = typecomp.BindType(iname)
590                  if type_info is not None:
591                      return [type_info]
592              except pythoncom.com_error:
593                  pass
594          return []
595  
596      def _GetTypeInfoCount_(self):
597          if self._typeinfos_ is None:
598              self._typeinfos_ = self._build_typeinfos_()
599          return len(self._typeinfos_)
600  
601      def _GetTypeInfo_(self, index, lcid):
602          if self._typeinfos_ is None:
603              self._typeinfos_ = self._build_typeinfos_()
604          if index < 0 or index >= len(self._typeinfos_):
605              raise COMException(scode=winerror.DISP_E_BADINDEX)
606          return 0, self._typeinfos_[index]
607  
608      def _allocnextdispid(self, last_dispid):
609          while 1:
610              last_dispid = last_dispid + 1
611              if (
612                  last_dispid not in self._dispid_to_func_
613                  and last_dispid not in self._dispid_to_get_
614                  and last_dispid not in self._dispid_to_put_
615              ):
616                  return last_dispid
617  
618      def _invokeex_(self, dispid, lcid, wFlags, args, kwArgs, serviceProvider):
619          ### note: lcid is being ignored...
620  
621          if wFlags & DISPATCH_METHOD:
622              try:
623                  funcname = self._dispid_to_func_[dispid]
624              except KeyError:
625                  if not wFlags & DISPATCH_PROPERTYGET:
626                      raise COMException(
627                          scode=winerror.DISP_E_MEMBERNOTFOUND
628                      )  # not found
629              else:
630                  try:
631                      func = getattr(self._obj_, funcname)
632                  except AttributeError:
633                      # May have a dispid, but that doesnt mean we have the function!
634                      raise COMException(scode=winerror.DISP_E_MEMBERNOTFOUND)
635                  # Should check callable here
636                  try:
637                      return func(*args)
638                  except TypeError as v:
639                      # Particularly nasty is "wrong number of args" type error
640                      # This helps you see what 'func' and 'args' actually is
641                      if str(v).find("arguments") >= 0:
642                          print(
643                              "** TypeError %s calling function %r(%r)" % (v, func, args)
644                          )
645                      raise
646  
647          if wFlags & DISPATCH_PROPERTYGET:
648              try:
649                  name = self._dispid_to_get_[dispid]
650              except KeyError:
651                  raise COMException(scode=winerror.DISP_E_MEMBERNOTFOUND)  # not found
652              retob = getattr(self._obj_, name)
653              if type(retob) == types.MethodType:  # a method as a property - call it.
654                  retob = retob(*args)
655              return retob
656  
657          if wFlags & (DISPATCH_PROPERTYPUT | DISPATCH_PROPERTYPUTREF):  ### correct?
658              try:
659                  name = self._dispid_to_put_[dispid]
660              except KeyError:
661                  raise COMException(scode=winerror.DISP_E_MEMBERNOTFOUND)  # read-only
662              # If we have a method of that name (ie, a property get function), and
663              # we have an equiv. property set function, use that instead.
664              if (
665                  type(getattr(self._obj_, name, None)) == types.MethodType
666                  and type(getattr(self._obj_, "Set" + name, None)) == types.MethodType
667              ):
668                  fn = getattr(self._obj_, "Set" + name)
669                  fn(*args)
670              else:
671                  # just set the attribute
672                  setattr(self._obj_, name, args[0])
673              return
674  
675          raise COMException(scode=winerror.E_INVALIDARG, desc="invalid wFlags")
676

678  class EventHandlerPolicy(DesignatedWrapPolicy):
679      """The default policy used by event handlers in the win32com.client package. 
680   
681      In addition to the base policy, this provides argument conversion semantics for 
682      params 
683        * dispatch params are converted to dispatch objects. 
684        * Unicode objects are converted to strings (1.5.2 and earlier) 
685   
686      NOTE: Later, we may allow the object to override this process?? 
687      """
688  
689      def _transform_args_(self, args, kwArgs, dispid, lcid, wFlags, serviceProvider):
690          ret = []
691          for arg in args:
692              arg_type = type(arg)
693              if arg_type == IDispatchType:
694                  import win32com.client
695  
696                  arg = win32com.client.Dispatch(arg)
697              elif arg_type == IUnknownType:
698                  try:
699                      import win32com.client
700  
701                      arg = win32com.client.Dispatch(
702                          arg.QueryInterface(pythoncom.IID_IDispatch)
703                      )
704                  except pythoncom.error:
705                      pass  # Keep it as IUnknown
706              ret.append(arg)
707          return tuple(ret), kwArgs
708  
709      def _invokeex_(self, dispid, lcid, wFlags, args, kwArgs, serviceProvider):
710          # transform the args.
711          args, kwArgs = self._transform_args_(
712              args, kwArgs, dispid, lcid, wFlags, serviceProvider
713          )
714          return DesignatedWrapPolicy._invokeex_(
715              self, dispid, lcid, wFlags, args, kwArgs, serviceProvider
716          )

719  class DynamicPolicy(BasicWrapPolicy):
720      """A policy which dynamically (ie, at run-time) determines public interfaces. 
721   
722      A dynamic policy is used to dynamically dispatch methods and properties to the 
723      wrapped object.  The list of objects and properties does not need to be known in 
724      advance, and methods or properties added to the wrapped object after construction 
725      are also handled. 
726   
727      The wrapped object must provide the following attributes: 
728   
729      _dynamic_ -- A method that will be called whenever an invoke on the object 
730             is called.  The method is called with the name of the underlying method/property 
731             (ie, the mapping of dispid to/from name has been resolved.)  This name property 
732             may also be '_value_' to indicate the default, and '_NewEnum' to indicate a new 
733             enumerator is requested. 
734   
735      """
736  
737      def _wrap_(self, object):
738          BasicWrapPolicy._wrap_(self, object)
739          if not hasattr(self._obj_, "_dynamic_"):
740              raise error("Object does not support Dynamic COM Policy")
741          self._next_dynamic_ = self._min_dynamic_ = 1000
742          self._dyn_dispid_to_name_ = {
743              DISPID_VALUE: "_value_",
744              DISPID_NEWENUM: "_NewEnum",
745          }
746  
747      def _getdispid_(self, name, fdex):
748          # TODO - Look at fdex flags.
749          lname = name.lower()
750          try:
751              return self._name_to_dispid_[lname]
752          except KeyError:
753              dispid = self._next_dynamic_ = self._next_dynamic_ + 1
754              self._name_to_dispid_[lname] = dispid
755              self._dyn_dispid_to_name_[dispid] = name  # Keep case in this map...
756              return dispid
757  
758      def _invoke_(self, dispid, lcid, wFlags, args):
759          return S_OK, -1, self._invokeex_(dispid, lcid, wFlags, args, None, None)
760  
761      def _invokeex_(self, dispid, lcid, wFlags, args, kwargs, serviceProvider):
762          ### note: lcid is being ignored...
763          ### note: kwargs is being ignored...
764          ### note: serviceProvider is being ignored...
765          ### there might be assigned DISPID values to properties, too...
766          try:
767              name = self._dyn_dispid_to_name_[dispid]
768          except KeyError:
769              raise COMException(
770                  scode=winerror.DISP_E_MEMBERNOTFOUND, desc="Member not found"
771              )
772          return self._obj_._dynamic_(name, lcid, wFlags, args)

778  def resolve_func(spec):
779      """Resolve a function by name 
780   
781      Given a function specified by 'module.function', return a callable object 
782      (ie, the function itself) 
783      """
784      try:
785          idx = spec.rindex(".")
786          mname = spec[:idx]
787          fname = spec[idx + 1 :]
788          # Dont attempt to optimize by looking in sys.modules,
789          # as another thread may also be performing the import - this
790          # way we take advantage of the built-in import lock.
791          module = _import_module(mname)
792          return getattr(module, fname)
793      except ValueError:  # No "." in name - assume in this module
794          return globals()[spec]
795  
796  
797  def call_func(spec, *args):
798      """Call a function specified by name. 
799   
800      Call a function specified by 'module.function' and return the result. 
801      """
802  
803      return resolve_func(spec)(*args)

806  def _import_module(mname):
807      """Import a module just like the 'import' statement. 
808   
809      Having this function is much nicer for importing arbitrary modules than 
810      using the 'exec' keyword.  It is more efficient and obvious to the reader. 
811      """
812      __import__(mname)
813      # Eeek - result of _import_ is "win32com" - not "win32com.a.b.c"
814      # Get the full module from sys.modules
815      return sys.modules[mname]
