knspace.py 
1    ''' 
2    Kivy Namespaces 
3    =============== 
4     
5    .. versionadded:: 1.9.1 
6     
7    .. warning:: 
8        This code is still experimental, and its API is subject to change in a 
9        future version. 
10    
11   The :class:`KNSpaceBehavior` `mixin <https://en.wikipedia.org/wiki/Mixin>`_ 
12   class provides namespace functionality for Kivy objects. It allows kivy objects 
13   to be named and then accessed using namespaces. 
14    
15   :class:`KNSpace` instances are the namespaces that store the named objects 
16   in Kivy :class:`~kivy.properties.ObjectProperty` instances. 
17   In addition, when inheriting from :class:`KNSpaceBehavior`, if the derived 
18   object is named, the name will automatically be added to the associated 
19   namespace and will point to a :attr:`~kivy.uix.widget.proxy_ref` of the 
20   derived object. 
21    
22   Basic examples 
23   -------------- 
24    
25   By default, there's only a single namespace: the :attr:`knspace` namespace. The 
26   simplest example is adding a widget to the namespace: 
27    
28   .. code-block:: python 
29    
30       from kivy.uix.behaviors.knspace import knspace 
31       widget = Widget() 
32       knspace.my_widget = widget 
33    
34   This adds a kivy :class:`~kivy.properties.ObjectProperty` with `rebind=True` 
35   and `allownone=True` to the :attr:`knspace` namespace with a property name 
36   `my_widget`. And the property now also points to this widget. 
37    
38   This can be done automatically with: 
39    
40   .. code-block:: python 
41    
42       class MyWidget(KNSpaceBehavior, Widget): 
43           pass 
44    
45       widget = MyWidget(knsname='my_widget') 
46    
47   Or in kv: 
48    
49   .. code-block:: kv 
50    
51       <MyWidget@KNSpaceBehavior+Widget> 
52    
53       MyWidget: 
54           knsname: 'my_widget' 
55    
56   Now, `knspace.my_widget` will point to that widget. 
57    
58   When one creates a second widget with the same name, the namespace will 
59   also change to point to the new widget. E.g.: 
60    
61   .. code-block:: python 
62    
63       widget = MyWidget(knsname='my_widget') 
64       # knspace.my_widget now points to widget 
65       widget2 = MyWidget(knsname='my_widget') 
66       # knspace.my_widget now points to widget2 
67    
68   Setting the namespace 
69   --------------------- 
70    
71   One can also create ones own namespace rather than using the default 
72   :attr:`knspace` by directly setting :attr:`KNSpaceBehavior.knspace`: 
73    
74   .. code-block:: python 
75    
76       class MyWidget(KNSpaceBehavior, Widget): 
77           pass 
78    
79       widget = MyWidget(knsname='my_widget') 
80       my_new_namespace = KNSpace() 
81       widget.knspace = my_new_namespace 
82    
83   Initially, `my_widget` is added to the default namespace, but when the widget's 
84   namespace is changed to `my_new_namespace`, the reference to `my_widget` is 
85   moved to that namespace. We could have also of course first set the namespace 
86   to `my_new_namespace` and then have named the widget `my_widget`, thereby 
87   avoiding the initial assignment to the default namespace. 
88    
89   Similarly, in kv: 
90    
91   .. code-block:: kv 
92    
93       <MyWidget@KNSpaceBehavior+Widget> 
94    
95       MyWidget: 
96           knspace: KNSpace() 
97           knsname: 'my_widget' 
98    
99   Inheriting the namespace 
100  ------------------------ 
101   
102  In the previous example, we directly set the namespace we wished to use. 
103  In the following example, we inherit it from the parent, so we only have to set 
104  it once: 
105   
106  .. code-block:: kv 
107   
108      <MyWidget@KNSpaceBehavior+Widget> 
109      <MyLabel@KNSpaceBehavior+Label> 
110   
111      <MyComplexWidget@MyWidget>: 
112          knsname: 'my_complex' 
113          MyLabel: 
114              knsname: 'label1' 
115          MyLabel: 
116              knsname: 'label2' 
117   
118  Then, we do: 
119   
120  .. code-block:: python 
121   
122      widget = MyComplexWidget() 
123      new_knspace = KNSpace() 
124      widget.knspace = new_knspace 
125   
126  The rule is that if no knspace has been assigned to a widget, it looks for a 
127  namespace in its parent and parent's parent and so on until it find one to 
128  use. If none are found, it uses the default :attr:`knspace`. 
129   
130  When `MyComplexWidget` is created, it still used the default namespace. 
131  However, when we assigned the root widget its new namespace, all its 
132  children switched to using that new namespace as well. So `new_knspace` now 
133  contains `label1` and `label2` as well as `my_complex`. 
134   
135  If we had first done: 
136   
137  .. code-block:: python 
138   
139      widget = MyComplexWidget() 
140      new_knspace = KNSpace() 
141      knspace.label1.knspace = knspace 
142      widget.knspace = new_knspace 
143   
144  Then `label1` would remain stored in the default :attr:`knspace` since it was 
145  directly set, but `label2` and `my_complex` would still be added to the new 
146  namespace. 
147   
148  One can customize the attribute used to search the parent tree by changing 
149  :attr:`KNSpaceBehavior.knspace_key`. If the desired knspace is not reachable 
150  through a widgets parent tree, e.g. in a popup that is not a widget's child, 
151  :attr:`KNSpaceBehavior.knspace_key` can be used to establish a different 
152  search order. 
153   
154  Accessing the namespace 
155  ----------------------- 
156   
157  As seen in the previous example, if not directly assigned, the namespace is 
158  found by searching the parent tree. Consequently, if a namespace was assigned 
159  further up the parent tree, all its children and below could access that 
160  namespace through their :attr:`KNSpaceBehavior.knspace` property. 
161   
162  This allows the creation of multiple widgets with identically given names 
163  if each root widget instance is assigned a new namespace. For example: 
164   
165  .. code-block:: kv 
166   
167      <MyComplexWidget@KNSpaceBehavior+Widget>: 
168          Label: 
169              text: root.knspace.pretty.text if root.knspace.pretty else '' 
170   
171      <MyPrettyWidget@KNSpaceBehavior+TextInput>: 
172          knsname: 'pretty' 
173          text: 'Hello' 
174   
175      <MyCompositeWidget@KNSpaceBehavior+BoxLayout>: 
176          MyComplexWidget 
177          MyPrettyWidget 
178   
179  Now, when we do: 
180   
181  .. code-block:: python 
182   
183      knspace1, knspace2 = KNSpace(), KNSpace() 
184      composite1 = MyCompositeWidget() 
185      composite1.knspace = knspace1 
186   
187      composite2 = MyCompositeWidget() 
188      composite2.knspace = knspace2 
189   
190      knspace1.pretty = "Here's the ladder, now fix the roof!" 
191      knspace2.pretty = "Get that raccoon off me!" 
192   
193  Because each of the `MyCompositeWidget` instances have a different namespace 
194  their children also use different namespaces. Consequently, the 
195  pretty and complex widgets of each instance will have different text. 
196   
197  Further, because both the namespace :class:`~kivy.properties.ObjectProperty` 
198  references, and :attr:`KNSpaceBehavior.knspace` have `rebind=True`, the 
199  text of the `MyComplexWidget` label is rebound to match the text of 
200  `MyPrettyWidget` when either the root's namespace changes or when the 
201  `root.knspace.pretty` property changes, as expected. 
202   
203  Forking a namespace 
204  ------------------- 
205   
206  Forking a namespace provides the opportunity to create a new namespace 
207  from a parent namespace so that the forked namespace will contain everything 
208  in the origin namespace, but the origin namespace will not have access to 
209  anything added to the forked namespace. 
210   
211  For example: 
212   
213  .. code-block:: python 
214   
215      child = knspace.fork() 
216      grandchild = child.fork() 
217   
218      child.label = Label() 
219      grandchild.button = Button() 
220   
221  Now label is accessible by both child and grandchild, but not by knspace. And 
222  button is only accessible by the grandchild but not by the child or by knspace. 
223  Finally, doing `grandchild.label = Label()` will leave `grandchild.label` 
224  and `child.label` pointing to different labels. 
225   
226  A motivating example is the example from above: 
227   
228  .. code-block:: kv 
229   
230      <MyComplexWidget@KNSpaceBehavior+Widget>: 
231          Label: 
232              text: root.knspace.pretty.text if root.knspace.pretty else '' 
233   
234      <MyPrettyWidget@KNSpaceBehavior+TextInput>: 
235          knsname: 'pretty' 
236          text: 'Hello' 
237   
238      <MyCompositeWidget@KNSpaceBehavior+BoxLayout>: 
239          knspace: 'fork' 
240          MyComplexWidget 
241          MyPrettyWidget 
242   
243  Notice the addition of `knspace: 'fork'`. This is identical to doing 
244  `knspace: self.knspace.fork()`. However, doing that would lead to infinite 
245  recursion as that kv rule would be executed recursively because `self.knspace` 
246  will keep on changing. However, allowing `knspace: 'fork'` cirumvents that. 
247  See :attr:`KNSpaceBehavior.knspace`. 
248   
249  Now, having forked, we just need to do: 
250   
251  .. code-block:: python 
252   
253      composite1 = MyCompositeWidget() 
254      composite2 = MyCompositeWidget() 
255   
256      composite1.knspace.pretty = "Here's the ladder, now fix the roof!" 
257      composite2.knspace.pretty = "Get that raccoon off me!" 
258   
259  Since by forking we automatically created a unique namespace for each 
260  `MyCompositeWidget` instance. 
261  '''

265  from kivy.event import EventDispatcher
266  from kivy.properties import StringProperty, ObjectProperty, AliasProperty
267  from kivy.context import register_context

270  class KNSpace(EventDispatcher):
271      '''Each :class:`KNSpace` instance is a namespace that stores the named Kivy 
272      objects associated with this namespace. Each named object is 
273      stored as the value of a Kivy :class:`~kivy.properties.ObjectProperty` of 
274      this instance whose property name is the object's given name. Both `rebind` 
275      and `allownone` are set to `True` for the property. 
276   
277      See :attr:`KNSpaceBehavior.knspace` for details on how a namespace is 
278      associated with a named object. 
279   
280      When storing an object in the namespace, the object's `proxy_ref` is 
281      stored if the object has such an attribute. 
282   
283      :Parameters: 
284   
285          `parent`: (internal) A :class:`KNSpace` instance or None. 
286              If specified, it's a parent namespace, in which case, the current 
287              namespace will have in its namespace all its named objects 
288              as well as the named objects of its parent and parent's parent 
289              etc. See :meth:`fork` for more details. 
290      '''
291  
292      parent = None
293      '''(internal) The parent namespace instance, :class:`KNSpace`, or None. See 
294      :meth:`fork`. 
295      '''
296      __has_applied = None
297  
298      keep_ref = False
299      '''Whether a direct reference should be kept to the stored objects. 
300      If ``True``, we use the direct object, otherwise we use 
301      :attr:`~kivy.uix.widget.proxy_ref` when present. 
302   
303      Defaults to False. 
304      '''
305  
306      def __init__(self, parent=None, keep_ref=False, **kwargs):
307          self.keep_ref = keep_ref
308          super(KNSpace, self).__init__(**kwargs)
309          self.parent = parent
310          self.__has_applied = set(self.properties().keys())
311  
312      def __setattr__(self, name, value):
313          prop = super(KNSpace, self).property(name, quiet=True)
314          has_applied = self.__has_applied
315          if prop is None:
316              if hasattr(self, name):
317                  super(KNSpace, self).__setattr__(name, value)
318              else:
319                  self.apply_property(
320                      **{name:
321                         ObjectProperty(None, rebind=True, allownone=True)}
322                  )
323                  if not self.keep_ref:
324                      value = getattr(value, 'proxy_ref', value)
325                  has_applied.add(name)
326                  super(KNSpace, self).__setattr__(name, value)
327          elif name not in has_applied:
328              self.apply_property(**{name: prop})
329              has_applied.add(name)
330              if not self.keep_ref:
331                  value = getattr(value, 'proxy_ref', value)
332              super(KNSpace, self).__setattr__(name, value)
333          else:
334              if not self.keep_ref:
335                  value = getattr(value, 'proxy_ref', value)
336              super(KNSpace, self).__setattr__(name, value)
337  
338      def __getattribute__(self, name):
339          if name in super(KNSpace, self).__getattribute__('__dict__'):
340              return super(KNSpace, self).__getattribute__(name)
341  
342          try:
343              value = super(KNSpace, self).__getattribute__(name)
344          except AttributeError:
345              parent = super(KNSpace, self).__getattribute__('parent')
346              if parent is None:
347                  raise AttributeError(name)
348              return getattr(parent, name)
349  
350          if value is not None:
351              return value
352  
353          parent = super(KNSpace, self).__getattribute__('parent')
354          if parent is None:
355              return None
356  
357          try:
358              return getattr(parent, name)  # if parent doesn't have it
359          except AttributeError:
360              return None
361  
362      def property(self, name, quiet=False):
363          # needs to overwrite EventDispatcher.property so kv lang will work
364          prop = super(KNSpace, self).property(name, quiet=True)
365          if prop is not None:
366              return prop
367  
368          prop = ObjectProperty(None, rebind=True, allownone=True)
369          self.apply_property(**{name: prop})
370          self.__has_applied.add(name)
371          return prop
372  
373      def fork(self):
374          '''Returns a new :class:`KNSpace` instance which will have access to 
375          all the named objects in the current namespace but will also have a 
376          namespace of its own that is unique to it. 
377   
378          For example: 
379   
380          .. code-block:: python 
381   
382              forked_knspace1 = knspace.fork() 
383              forked_knspace2 = knspace.fork() 
384   
385          Now, any names added to `knspace` will be accessible by the 
386          `forked_knspace1` and `forked_knspace2` namespaces by the normal means. 
387          However, any names added to `forked_knspace1` will not be accessible 
388          from `knspace` or `forked_knspace2`. Similar for `forked_knspace2`. 
389          '''
390          return KNSpace(parent=self)

393  class KNSpaceBehavior(object):
394      '''Inheriting from this class allows naming of the inherited objects, which 
395      are then added to the associated namespace :attr:`knspace` and accessible 
396      through it. 
397   
398      Please see the :mod:`knspace behaviors module <kivy.uix.behaviors.knspace>` 
399      documentation for more information. 
400      '''
401  
402      _knspace = ObjectProperty(None, allownone=True)
403      _knsname = StringProperty('')
404      __last_knspace = None
405      __callbacks = None
406  
407      def __init__(self, knspace=None, **kwargs):
408          self.knspace = knspace
409          super(KNSpaceBehavior, self).__init__(**kwargs)
410  
411      def __knspace_clear_callbacks(self, *largs):
412          for obj, name, uid in self.__callbacks:
413              obj.unbind_uid(name, uid)
414          last = self.__last_knspace
415          self.__last_knspace = self.__callbacks = None
416  
417          assert self._knspace is None
418          assert last
419  
420          new = self.__set_parent_knspace()
421          if new is last:
422              return
423          self.property('_knspace').dispatch(self)
424  
425          name = self.knsname
426          if not name:
427              return
428  
429          if getattr(last, name) == self:
430              setattr(last, name, None)
431  
432          if new:
433              setattr(new, name, self)
434          else:
435              raise ValueError('Object has name "{}", but no namespace'.
436                               format(name))
437  
438      def __set_parent_knspace(self):
439          callbacks = self.__callbacks = []
440          fbind = self.fbind
441          append = callbacks.append
442          parent_key = self.knspace_key
443          clear = self.__knspace_clear_callbacks
444  
445          append((self, 'knspace_key', fbind('knspace_key', clear)))
446          if not parent_key:
447              self.__last_knspace = knspace
448              return knspace
449  
450          append((self, parent_key, fbind(parent_key, clear)))
451          parent = getattr(self, parent_key, None)
452          while parent is not None:
453              fbind = parent.fbind
454  
455              parent_knspace = getattr(parent, 'knspace', 0)
456              if parent_knspace != 0:
457                  append((parent, 'knspace', fbind('knspace', clear)))
458                  self.__last_knspace = parent_knspace
459                  return parent_knspace
460  
461              append((parent, parent_key, fbind(parent_key, clear)))
462              new_parent = getattr(parent, parent_key, None)
463              if new_parent is parent:
464                  break
465              parent = new_parent
466          self.__last_knspace = knspace
467          return knspace
468  
469      def _get_knspace(self):
470          _knspace = self._knspace
471          if _knspace is not None:
472              return _knspace
473  
474          if self.__callbacks is not None:
475              return self.__last_knspace
476  
477          # we only get here if we never accessed our knspace
478          return self.__set_parent_knspace()
479  
480      def _set_knspace(self, value):
481          if value is self._knspace:
482              return
483  
484          knspace = self._knspace or self.__last_knspace
485          name = self.knsname
486          if name and knspace and getattr(knspace, name) == self:
487              setattr(knspace, name, None)  # reset old namespace
488  
489          if value == 'fork':
490              if not knspace:
491                  knspace = self.knspace  # get parents in case we haven't before
492              if knspace:
493                  value = knspace.fork()
494              else:
495                  raise ValueError('Cannot fork with no namespace')
496  
497          for obj, prop_name, uid in self.__callbacks or []:
498              obj.unbind_uid(prop_name, uid)
499          self.__last_knspace = self.__callbacks = None
500  
501          if name:
502              if value is None:  # if None, first update the recursive knspace
503                  knspace = self.__set_parent_knspace()
504                  if knspace:
505                      setattr(knspace, name, self)
506                  self._knspace = None  # cause a kv trigger
507              else:
508                  setattr(value, name, self)
509                  knspace = self._knspace = value
510  
511              if not knspace:
512                  raise ValueError('Object has name "{}", but no namespace'.
513                                   format(name))
514          else:
515              if value is None:
516                  self.__set_parent_knspace()  # update before trigger below
517              self._knspace = value
518  
519      knspace = AliasProperty(
520          _get_knspace, _set_knspace, bind=('_knspace', ), cache=False,
521          rebind=True, allownone=True)
522      '''The namespace instance, :class:`KNSpace`, associated with this widget. 
523      The :attr:`knspace` namespace stores this widget when naming this widget 
524      with :attr:`knsname`. 
525   
526      If the namespace has been set with a :class:`KNSpace` instance, e.g. with 
527      `self.knspace = KNSpace()`, then that instance is returned (setting with 
528      `None` doesn't count). Otherwise, if :attr:`knspace_key` is not None, we 
529      look for a namespace to use in the object that is stored in the property 
530      named :attr:`knspace_key`, of this instance. I.e. 
531      `object = getattr(self, self.knspace_key)`. 
532   
533      If that object has a knspace property, then we return its value. Otherwise, 
534      we go further up, e.g. with `getattr(object, self.knspace_key)` and look 
535      for its `knspace` property. 
536   
537      Finally, if we reach a value of `None`, or :attr:`knspace_key` was `None`, 
538      the default :attr:`~kivy.uix.behaviors.knspace.knspace` namespace is 
539      returned. 
540   
541      If :attr:`knspace` is set to the string `'fork'`, the current namespace 
542      in :attr:`knspace` will be forked with :meth:`KNSpace.fork` and the 
543      resulting namespace will be assigned to this instance's :attr:`knspace`. 
544      See the module examples for a motivating example. 
545   
546      Both `rebind` and `allownone` are `True`. 
547      '''
548  
549      knspace_key = StringProperty('parent', allownone=True)
550      '''The name of the property of this instance, to use to search upwards for 
551      a namespace to use by this instance. Defaults to `'parent'` so that we'll 
552      search the parent tree. See :attr:`knspace`. 
553   
554      When `None`, we won't search the parent tree for the namespace. 
555      `allownone` is `True`. 
556      '''
557  
558      def _get_knsname(self):
559          return self._knsname
560  
561      def _set_knsname(self, value):
562          old_name = self._knsname
563          knspace = self.knspace
564          if old_name and knspace and getattr(knspace, old_name) == self:
565              setattr(knspace, old_name, None)
566  
567          self._knsname = value
568          if value:
569              if knspace:
570                  setattr(knspace, value, self)
571              else:
572                  raise ValueError('Object has name "{}", but no namespace'.
573                                   format(value))
574  
575      knsname = AliasProperty(
576          _get_knsname, _set_knsname, bind=('_knsname', ), cache=False)
577      '''The name given to this instance. If named, the name will be added to the 
578      associated :attr:`knspace` namespace, which will then point to the 
579      `proxy_ref` of this instance. 
580   
581      When named, one can access this object by e.g. self.knspace.name, where 
582      `name` is the given name of this instance. See :attr:`knspace` and the 
583      module description for more details. 
584      '''
585  
586  
587  knspace = register_context('knspace', KNSpace)
588  '''The default :class:`KNSpace` namespace. See :attr:`KNSpaceBehavior.knspace` 
589  for more details.
