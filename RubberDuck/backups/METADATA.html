32   # Mapview
33   
34   [![Github Build Status](https://github.com/kivy-garden/mapview/workflows/Tests/badge.svg)](https://github.com/kivy-garden/mapview/actions?query=workflow%3ATests)
35   [![Build Status](https://travis-ci.com/kivy-garden/mapview.svg?branch=develop)](https://travis-ci.com/kivy-garden/mapview)
36   [![Coverage Status](https://coveralls.io/repos/github/kivy-garden/mapview/badge.svg?branch=develop)](https://coveralls.io/github/kivy-garden/mapview?branch=develop)
37   [![PyPI version](https://badge.fury.io/py/mapview.svg)](https://badge.fury.io/py/mapview)
38   
39   Mapview is a Kivy widget for displaying interactive maps. It has been
40   designed with lot of inspirations of
41   [Libchamplain](https://wiki.gnome.org/Projects/libchamplain) and
42   [Leaflet](http://leafletjs.com/).
43   
44   The goal of this widget is to be a replacement of Google Maps widget,
45   even if this one works very well, it just works on Android with Kivy.
46   I wanted a map widget that can support custom map, and designed with
47   the latests state-of-the-art Kivy's methods.
48   
49   ![ScreenShot](https://cloud.githubusercontent.com/assets/37904/22764226/925c93ce-ee69-11e6-90ed-88123bfa731f.png)
50   
51   ![Marker clustering](https://cloud.githubusercontent.com/assets/37904/22764225/92516f12-ee69-11e6-99d5-6346e302766d.png)
52   
53   # Features
54   
55   * native multitouch (one for translate, many for translate and zoom)
56   * asynchronous downloading
57   * avoided GPU limitation / float precisions issues on tiles coordinates
58   * marker support
59   * blazing fast!
60   * supports Z/X/Y providers by default with `MapSource`
61   * supports [.mbtiles](http://mbtiles.org) via `MBTilesMapSource`
62   * supports marker clustering, via `ClusteredMarkerLayer`
63   
64   # Requirements
65   
66   It requires the `concurrent.futures` and `requests`.
67   If you use it on Android / iOS, don't forget to add `openssl` as a requirements,
68   otherwise you'll have an issue when importing `urllib3` from `requests`.
69

76   # Usage
77   
78   This widget can be either used within Python or Kv. That's said, not
79   everything can be done in Kv, to prevent too much computing.
80   
81   ```python
82   from kivy_garden.mapview import MapView
83   from kivy.app import App
84   
85   class MapViewApp(App):
86       def build(self):
87           mapview = MapView(zoom=11, lat=50.6394, lon=3.057)
88           return mapview
89   
90   MapViewApp().run()
91   ```
