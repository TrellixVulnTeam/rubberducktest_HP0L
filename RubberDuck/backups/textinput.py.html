2    ''' 
3    Text Input 
4    ========== 
5     
6    .. versionadded:: 1.0.4 
7     
8    .. image:: images/textinput-mono.jpg 
9    .. image:: images/textinput-multi.jpg 
10    
11   The :class:`TextInput` widget provides a box for editable plain text. 
12    
13   Unicode, multiline, cursor navigation, selection and clipboard features 
14   are supported. 
15    
16   The :class:`TextInput` uses two different coordinate systems: 
17    
18   * (x, y) - coordinates in pixels, mostly used for rendering on screen. 
19   * (col, row) - cursor index in characters / lines, used for selection 
20     and cursor movement. 
21    
22    
23   Usage example 
24   ------------- 
25    
26   To create a multiline :class:`TextInput` (the 'enter' key adds a new line):: 
27    
28       from kivy.uix.textinput import TextInput 
29       textinput = TextInput(text='Hello world') 
30    
31   To create a singleline :class:`TextInput`, set the :class:`TextInput.multiline` 
32   property to False (the 'enter' key will defocus the TextInput and emit an 
33   :meth:`TextInput.on_text_validate` event):: 
34    
35       def on_enter(instance, value): 
36           print('User pressed enter in', instance) 
37    
38       textinput = TextInput(text='Hello world', multiline=False) 
39       textinput.bind(on_text_validate=on_enter) 
40    
41   The textinput's text is stored in its :attr:`TextInput.text` property. To run a 
42   callback when the text changes:: 
43    
44       def on_text(instance, value): 
45           print('The widget', instance, 'have:', value) 
46    
47       textinput = TextInput() 
48       textinput.bind(text=on_text) 
49    
50   You can set the :class:`focus <kivy.uix.behaviors.FocusBehavior>` to a 
51   Textinput, meaning that the input box will be highlighted and keyboard focus 
52   will be requested:: 
53    
54       textinput = TextInput(focus=True) 
55    
56   The textinput is defocused if the 'escape' key is pressed, or if another 
57   widget requests the keyboard. You can bind a callback to the focus property to 
58   get notified of focus changes:: 
59    
60       def on_focus(instance, value): 
61           if value: 
62               print('User focused', instance) 
63           else: 
64               print('User defocused', instance) 
65    
66       textinput = TextInput() 
67       textinput.bind(focus=on_focus) 
68    
69   See :class:`~kivy.uix.behaviors.FocusBehavior`, from which the 
70   :class:`TextInput` inherits, for more details. 
71    
72    
73   Selection 
74   --------- 
75    
76   The selection is automatically updated when the cursor position changes. 
77   You can get the currently selected text from the 
78   :attr:`TextInput.selection_text` property. 
79    
80   Filtering 
81   --------- 
82    
83   You can control which text can be added to the :class:`TextInput` by 
84   overwriting :meth:`TextInput.insert_text`. Every string that is typed, pasted 
85   or inserted by any other means into the :class:`TextInput` is passed through 
86   this function. By overwriting it you can reject or change unwanted characters. 
87    
88   For example, to write only in capitalized characters:: 
89    
90       class CapitalInput(TextInput): 
91    
92           def insert_text(self, substring, from_undo=False): 
93               s = substring.upper() 
94               return super(CapitalInput, self).insert_text(s,\ 
95    from_undo=from_undo) 
96    
97   Or to only allow floats (0 - 9 and a single period):: 
98    
99       class FloatInput(TextInput): 
100   
101          pat = re.compile('[^0-9]') 
102          def insert_text(self, substring, from_undo=False): 
103              pat = self.pat 
104              if '.' in self.text: 
105                  s = re.sub(pat, '', substring) 
106              else: 
107                  s = '.'.join([re.sub(pat, '', s) for s in\ 
108   substring.split('.', 1)]) 
109              return super(FloatInput, self).insert_text(s, from_undo=from_undo) 
110   
111  Default shortcuts 
112  ----------------- 
113   
114  =============== ======================================================== 
115     Shortcuts    Description 
116  --------------- -------------------------------------------------------- 
117  Left            Move cursor to left 
118  Right           Move cursor to right 
119  Up              Move cursor to up 
120  Down            Move cursor to down 
121  Home            Move cursor at the beginning of the line 
122  End             Move cursor at the end of the line 
123  PageUp          Move cursor to 3 lines before 
124  PageDown        Move cursor to 3 lines after 
125  Backspace       Delete the selection or character before the cursor 
126  Del             Delete the selection of character after the cursor 
127  Shift + <dir>   Start a text selection. Dir can be Up, Down, Left or 
128                  Right 
129  Control + c     Copy selection 
130  Control + x     Cut selection 
131  Control + v     Paste clipboard content 
132  Control + a     Select all the content 
133  Control + z     undo 
134  Control + r     redo 
135  =============== ======================================================== 
136   
137  .. note:: 
138      To enable Emacs-style keyboard shortcuts, you can use 
139      :class:`~kivy.uix.behaviors.emacs.EmacsBehavior`.

405   
406      :Events: 
407          `on_text_validate` 
408              Fired only in multiline=False mode when the user hits 'enter'. 
409              This will also unfocus the textinput. 
410          `on_double_tap` 
411              Fired when a double tap happens in the text input. The default 
412              behavior selects the text around the cursor position. More info at 
413              :meth:`on_double_tap`. 
414          `on_triple_tap` 
415              Fired when a triple tap happens in the text input. The default 
416              behavior selects the line around the cursor position. More info at 
417              :meth:`on_triple_tap`.

419              Fired when four fingers are touching the text input. The default 
420              behavior selects the whole text. More info at

423      .. warning:: 
424          When changing a :class:`TextInput` property that requires re-drawing, 
425          e.g. modifying the :attr:`text`, the updates occur on the next 
426          clock cycle and not instantly. This might cause any changes to the 
427          :class:`TextInput` that occur between the modification and the next 
428          cycle to be ignored, or to use previous values. For example, after 
429          a update to the :attr:`text`, changing the cursor in the same clock 
430          frame will move it using the previous text and will likely end up in an

432          on the next clock cycle using

588      def cursor_index(self, cursor=None):
589          '''Return the cursor index in the text/value. 
590          '''
591          if not cursor:
592              cursor = self.cursor
593          try:
594              l = self._lines
595              if len(l) == 0:
596                  return 0
597              lf = self._lines_flags
598              index, cr = cursor
599              for row in range(cr):
600                  if row >= len(l):
601                      continue
602                  index += len(l[row])
603                  if lf[row] & FL_IS_LINEBREAK:
604                      index += 1
605              if lf[cr] & FL_IS_LINEBREAK:
606                  index += 1
607              return index
608          except IndexError:
609              return 0

626      def get_cursor_from_index(self, index):
627          '''Return the (col, row) of the cursor from text index. 
628          '''
629          index = boundary(index, 0, len(self.text))
630          if index <= 0:
631              return 0, 0
632          lf = self._lines_flags
633          l = self._lines
634          i = 0
635          for row in range(len(l)):
636              ni = i + len(l[row])
637              if lf[row] & FL_IS_LINEBREAK:
638                  ni += 1
639                  i += 1
640              if ni >= index:
641                  return index - i, row
642              i = ni
643          return index, row

675      def _auto_indent(self, substring):
676          index = self.cursor_index()
677          if index > 0:
678              _text = self.text
679              line_start = _text.rfind('\n', 0, index)
680              if line_start > -1:
681                  line = _text[line_start + 1:index]
682                  indent = self.re_indent.match(line).group()
683                  substring += indent
684          return substring

686      def insert_text(self, substring, from_undo=False):
687          '''Insert new text at the current cursor position. Override this 
688          function in order to pre-process text for input validation. 
689          '''
690          if self.readonly or not substring or not self._lines:
691              return
692  
693          if isinstance(substring, bytes):
694              substring = substring.decode('utf8')
695  
696          if self.replace_crlf:
697              substring = substring.replace(u'\r\n', u'\n')
698  
699          self._hide_handles(EventLoop.window)
700  
701          if not from_undo and self.multiline and self.auto_indent \
702                  and substring == u'\n':
703              substring = self._auto_indent(substring)
704  
705          mode = self.input_filter
706          if mode not in (None, 'int', 'float'):
707              substring = mode(substring, from_undo)
708              if not substring:
709                  return
710  
711          cc, cr = self.cursor
712          sci = self.cursor_index
713          ci = sci()
714          text = self._lines[cr]
715          len_str = len(substring)
716          new_text = text[:cc] + substring + text[cc:]
717          if mode is not None:
718              if mode == 'int':
719                  if not re.match(self._insert_int_pat, new_text):
720                      return
721              elif mode == 'float':
722                  if not re.match(self._insert_float_pat, new_text):
723                      return
724          self._set_line_text(cr, new_text)
725  
726          wrap = (self._get_text_width(
727              new_text,
728              self.tab_width,
729              self._label_cached) > (self.width - self.padding[0] -
730                                     self.padding[2]))
731          if len_str > 1 or substring == u'\n' or wrap:
732              # Avoid refreshing text on every keystroke.
733              # Allows for faster typing of text when the amount of text in
734              # TextInput gets large.
735  
736              start, finish, lines,\
737                  lineflags, len_lines = self._get_line_from_cursor(cr, new_text)
738              # calling trigger here could lead to wrong cursor positioning
739              # and repeating of text when keys are added rapidly in a automated
740              # fashion. From Android Keyboard for example.
741              self._refresh_text_from_property('insert', start, finish, lines,
742                                               lineflags, len_lines)
743  
744          self.cursor = self.get_cursor_from_index(ci + len_str)
745          # handle undo and redo
746          self._set_unredo_insert(ci, ci + len_str, substring, from_undo)

748      def _get_line_from_cursor(self, start, new_text):
749          # get current paragraph from cursor position
750          finish = start
751          lines = self._lines
752          linesflags = self._lines_flags
753          if start and not linesflags[start]:
754              start -= 1
755              new_text = u''.join((lines[start], new_text))
756          try:
757              while not linesflags[finish + 1]:
758                  new_text = u''.join((new_text, lines[finish + 1]))
759                  finish += 1
760          except IndexError:
761              pass
762          lines, lineflags = self._split_smart(new_text)
763          len_lines = max(1, len(lines))
764          return start, finish, lines, lineflags, len_lines

783      def do_redo(self):
784          '''Do redo operation. 
785   
786          .. versionadded:: 1.3.0 
787   
788          This action re-does any command that has been un-done by 
789          do_undo/ctrl+z. This function is automatically called when 
790          `ctrl+r` keys are pressed. 
791          '''
792          try:
793              x_item = self._redo.pop()
794              undo_type = x_item['undo_command'][0]
795              _get_cusror_from_index = self.get_cursor_from_index
796  
797              if undo_type == 'insert':
798                  ci, substring = x_item['redo_command']
799                  self.cursor = _get_cusror_from_index(ci)
800                  self.insert_text(substring, True)
801              elif undo_type == 'bkspc':
802                  self.cursor = _get_cusror_from_index(x_item['redo_command'])
803                  self.do_backspace(from_undo=True)
804              elif undo_type == 'shiftln':
805                  direction, rows, cursor = x_item['redo_command'][1:]
806                  self._shift_lines(direction, rows, cursor, True)
807              else:
808                  # delsel
809                  ci, sci = x_item['redo_command']
810                  self._selection_from = ci
811                  self._selection_to = sci
812                  self._selection = True
813                  self.delete_selection(True)
814                  self.cursor = _get_cusror_from_index(ci)
815              self._undo.append(x_item)
816          except IndexError:
817              # reached at top of undo list
818              pass

827          This function is automatically called when `ctrl+z` keys are pressed.

847                  # delsel
848                  substring = x_item['undo_command'][2:][0]

855      def do_backspace(self, from_undo=False, mode='bkspc'):
856          '''Do backspace operation from the current cursor position. 
857          This action might do several things: 
858   
859              - removing the current selection if available. 
860              - removing the previous char and move the cursor back. 
861              - do nothing, if we are at the start. 
862   
863          '''
864          # IME system handles its own backspaces
865          if self.readonly or self._ime_composition:
866              return
867          cc, cr = self.cursor
868          _lines = self._lines
869          text = _lines[cr]
870          cursor_index = self.cursor_index()
871          text_last_line = _lines[cr - 1]
872  
873          if cc == 0 and cr == 0:
874              return
875          _lines_flags = self._lines_flags
876          start = cr
877          if cc == 0:
878              substring = u'\n' if _lines_flags[cr] else u' '
879              new_text = text_last_line + text
880              self._set_line_text(cr - 1, new_text)
881              self._delete_line(cr)
882              start = cr - 1
883          else:
884              # ch = text[cc-1]
885              substring = text[cc - 1]
886              new_text = text[:cc - 1] + text[cc:]
887              self._set_line_text(cr, new_text)
888  
889          # refresh just the current line instead of the whole text
890          start, finish, lines, lineflags, len_lines =\
891              self._get_line_from_cursor(start, new_text)
892          # avoid trigger refresh, leads to issue with
893          # keys/text send rapidly through code.
894          self._refresh_text_from_property('del', start, finish, lines,
895                                           lineflags, len_lines)
896  
897          self.cursor = self.get_cursor_from_index(cursor_index - 1)
898          # handle undo and redo
899          self._set_undo_redo_bkspc(
900              cursor_index,
901              cursor_index - 1,
902              substring, from_undo)

916      def _move_cursor_word_left(self, index=None):
917          pos = index or self.cursor_index()
918          if pos == 0:
919              return self.cursor
920          lines = self._lines
921          col, row = self.get_cursor_from_index(pos)
922          if col == 0:
923              row -= 1
924              col = len(lines[row])
925          while True:
926              matches = list(self._re_whitespace.finditer(lines[row], 0, col))
927              if not matches:
928                  if col == 0:
929                      if row == 0:
930                          return 0, 0
931                      row -= 1
932                      col = len(lines[row])
933                      continue
934                  return 0, row
935              match = matches[-1]
936              mpos = match.end()
937              if mpos == col:
938                  if len(matches) > 1:
939                      match = matches[-2]
940                      mpos = match.end()
941                  else:
942                      if match.start() == 0:
943                          if row == 0:
944                              return 0, 0
945                          row -= 1
946                          col = len(lines[row])
947                          continue
948                      return 0, row
949              col = mpos
950              return col, row

952      def _move_cursor_word_right(self, index=None):
953          pos = index or self.cursor_index()
954          col, row = self.get_cursor_from_index(pos)
955          lines = self._lines
956          mrow = len(lines) - 1
957          if row == mrow and col == len(lines[row]):
958              return col, row
959          if col == len(lines[row]):
960              row += 1
961              col = 0
962          while True:
963              matches = list(self._re_whitespace.finditer(lines[row], col))
964              if not matches:
965                  if col == len(lines[row]):
966                      if row == mrow:
967                          return col, row
968                      row += 1
969                      col = 0
970                      continue
971                  return len(lines[row]), row
972              match = matches[0]
973              mpos = match.start()
974              if mpos == col:
975                  if len(matches) > 1:
976                      match = matches[1]
977                      mpos = match.start()
978                  else:
979                      if match.end() == len(lines[row]):
980                          if row == mrow:
981                              return col, row
982                          row += 1
983                          col = 0
984                          continue
985                      return len(lines[row]), row
986              col = mpos
987              return col, row

999      def _expand_rows(self, rfrom, rto=None):
1000         if rto is None or rto == rfrom:
1001             rto = rfrom + 1
1002         lines = self._lines
1003         flags = list(reversed(self._lines_flags))
1004         while rfrom > 0 and not (flags[rfrom - 1] & FL_IS_NEWLINE):
1005             rfrom -= 1
1006         rmax = len(lines) - 1
1007         while 0 < rto < rmax and not (flags[rto - 1] & FL_IS_NEWLINE):
1008             rto += 1
1009         return max(0, rfrom), min(rmax, rto)

1089     def do_cursor_movement(self, action, control=False, alt=False):
1090         '''Move the cursor relative to its current position. 
1091         Action can be one of : 
1092  
1093             - cursor_left: move the cursor to the left 
1094             - cursor_right: move the cursor to the right 
1095             - cursor_up: move the cursor on the previous line 
1096             - cursor_down: move the cursor on the next line 
1097             - cursor_home: move the cursor at the start of the current line 
1098             - cursor_end: move the cursor at the end of current line 
1099             - cursor_pgup: move one "page" before 
1100             - cursor_pgdown: move one "page" after 
1101  
1102         In addition, the behavior of certain actions can be modified: 
1103  
1104             - control + cursor_left: move the cursor one word to the left 
1105             - control + cursor_right: move the cursor one word to the right 
1106             - control + cursor_up: scroll up one line 
1107             - control + cursor_down: scroll down one line 
1108             - control + cursor_home: go to beginning of text 
1109             - control + cursor_end: go to end of text 
1110             - alt + cursor_up: shift line(s) up 
1111             - alt + cursor_down: shift line(s) down 
1112  
1113         .. versionchanged:: 1.9.1 
1114  
1115         '''
1116         if not self._lines:
1117             return
1118         pgmove_speed = int(self.height /
1119             (self.line_height + self.line_spacing) - 1)
1120         col, row = self.cursor
1121         if action == 'cursor_up':
1122             if self.multiline and control:
1123                 self.scroll_y = max(0, self.scroll_y - self.line_height)
1124             elif not self.readonly and self.multiline and alt:
1125                 self._shift_lines(-1)
1126                 return
1127             else:
1128                 row = max(row - 1, 0)
1129                 col = min(len(self._lines[row]), col)
1130         elif action == 'cursor_down':
1131             if self.multiline and control:
1132                 maxy = self.minimum_height - self.height
1133                 self.scroll_y = max(0, min(maxy,
1134                                            self.scroll_y + self.line_height))
1135             elif not self.readonly and self.multiline and alt:
1136                 self._shift_lines(1)
1137                 return
1138             else:
1139                 row = min(row + 1, len(self._lines) - 1)
1140                 col = min(len(self._lines[row]), col)
1141         elif action == 'cursor_home':
1142             col = 0
1143             if control:
1144                 row = 0
1145         elif action == 'cursor_end':
1146             if control:
1147                 row = len(self._lines) - 1
1148             col = len(self._lines[row])
1149         elif action == 'cursor_pgup':
1150             row = max(0, row - pgmove_speed)
1151             col = min(len(self._lines[row]), col)
1152         elif action == 'cursor_pgdown':
1153             row = min(row + pgmove_speed, len(self._lines) - 1)
1154             col = min(len(self._lines[row]), col)
1155         elif (self._selection and self._selection_finished and
1156                 self._selection_from < self._selection_to and
1157                 action == 'cursor_left'):
1158             current_selection_to = self._selection_to
1159             while self._selection_from != current_selection_to:
1160                 current_selection_to -= 1
1161                 if col:
1162                     col -= 1
1163                 else:
1164                     row -= 1
1165                     col = len(self._lines[row])
1166         elif (self._selection and self._selection_finished and
1167                 self._selection_from > self._selection_to and
1168                 action == 'cursor_right'):
1169             current_selection_to = self._selection_to
1170             while self._selection_from != current_selection_to:
1171                 current_selection_to += 1
1172                 if len(self._lines[row]) > col:
1173                     col += 1
1174                 else:
1175                     row += 1
1176                     col = 0
1177 
1178         elif action == 'cursor_left':
1179             if not self.password and control:
1180                 col, row = self._move_cursor_word_left()
1181             else:
1182                 if col == 0:
1183                     if row:
1184                         row -= 1
1185                         col = len(self._lines[row])
1186                 else:
1187                     col, row = col - 1, row
1188         elif action == 'cursor_right':
1189             if not self.password and control:
1190                 col, row = self._move_cursor_word_right()
1191             else:
1192                 if col == len(self._lines[row]):
1193                     if row < len(self._lines) - 1:
1194                         col = 0
1195                         row += 1
1196                 else:
1197                     col, row = col + 1, row
1198 
1199         dont_move_cursor = control and action in ['cursor_up', 'cursor_down']
1200         if dont_move_cursor:
1201             self._trigger_update_graphics()
1202         else:
1203             self.cursor = (col, row)

1205     def get_cursor_from_xy(self, x, y):
1206         '''Return the (col, row) of the cursor from an (x, y) position. 
1207         '''
1208         padding_left = self.padding[0]
1209         padding_top = self.padding[1]
1210         l = self._lines
1211         dy = self.line_height + self.line_spacing
1212         cx = x - self.x
1213         scrl_y = self.scroll_y
1214         scrl_x = self.scroll_x
1215         scrl_y = scrl_y / dy if scrl_y > 0 else 0
1216         cy = (self.top - padding_top + scrl_y * dy) - y
1217         cy = int(boundary(round(cy / dy - 0.5), 0, len(l) - 1))
1218         _get_text_width = self._get_text_width
1219         _tab_width = self.tab_width
1220         _label_cached = self._label_cached
1221         # Offset for horizontal text alignment
1222         xoff = 0

1224         base_dir = self.base_direction or self._resolved_base_dir
1225         auto_halign_r = halign == 'auto' and base_dir and 'rtl' in base_dir
1226         if halign == 'center':
1227             viewport_width = self.width - padding_left - self.padding[2]  # _r
1228             xoff = int((viewport_width - self._get_row_width(cy)) / 2)

1231             xoff = viewport_width - self._get_row_width(cy)

1234                   _get_text_width(l[cy][i], _tab_width, _label_cached) * 0.6 +\
1235                   padding_left > cx + scrl_x:
1236                 cx = i
1237                 break

1880     def _refresh_text(self, text, *largs):
1881         # Refresh all the lines from a new text.
1882         # By using cache in internal functions, this method should be fast.
1883         mode = 'all'
1884         if len(largs) > 1:
1885             mode, start, finish, _lines, _lines_flags, len_lines = largs
1886             # start = max(0, start)
1887             cursor = None
1888         else:
1889             cursor = self.cursor_index()
1890             _lines, self._lines_flags = self._split_smart(text)
1891         _lines_labels = []
1892         _line_rects = []
1893         _create_label = self._create_line_label
1894 
1895         for x in _lines:
1896             lbl = _create_label(x)
1897             _lines_labels.append(lbl)
1898             _line_rects.append(Rectangle(size=lbl.size))
1899 
1900         if mode == 'all':
1901             self._lines_labels = _lines_labels
1902             self._lines_rects = _line_rects
1903             self._lines[:] = _lines
1904         elif mode == 'del':
1905             if finish > start:
1906                 self._insert_lines(start,
1907                                    finish if start == finish else (finish + 1),
1908                                    len_lines, _lines_flags,
1909                                    _lines, _lines_labels, _line_rects)
1910         elif mode == 'insert':
1911             self._insert_lines(
1912                 start,
1913                 finish if (start == finish and not len_lines)
1914                 else (finish + 1),
1915                 len_lines, _lines_flags, _lines, _lines_labels,
1916                 _line_rects)

1918         min_line_ht = self._label_cached.get_extents('_')[1]
1919         # with markup texture can be of height `1`
1920         self.line_height = max(_lines_labels[0].height, min_line_ht)

2320     def _split_smart(self, text):
2321         # Do a "smart" split. If autowidth or autosize is set,
2322         # we are not doing smart split, just a split on line break.
2323         # Otherwise, we are trying to split as soon as possible, to prevent
2324         # overflow on the widget.
2325 
2326         # depend of the options, split the text on line, or word
2327         if not self.multiline:
2328             lines = text.split(u'\n')
2329             lines_flags = [0] + [FL_IS_LINEBREAK] * (len(lines) - 1)
2330             return lines, lines_flags
2331 
2332         # no autosize, do wordwrap.
2333         x = flags = 0
2334         line = []
2335         lines = []
2336         lines_flags = []
2337         _join = u''.join
2338         lines_append, lines_flags_append = lines.append, lines_flags.append
2339         padding_left = self.padding[0]
2340         padding_right = self.padding[2]
2341         width = self.width - padding_left - padding_right
2342         text_width = self._get_text_width
2343         _tab_width, _label_cached = self.tab_width, self._label_cached
2344 
2345         # try to add each word on current line.
2346         for word in self._tokenize(text):
2347             is_newline = (word == u'\n')
2348             w = text_width(word, _tab_width, _label_cached)
2349             # if we have more than the width, or if it's a newline,
2350             # push the current line, and create a new one
2351             if (x + w > width and line) or is_newline:
2352                 lines_append(_join(line))
2353                 lines_flags_append(flags)
2354                 flags = 0
2355                 line = []
2356                 x = 0
2357             if is_newline:
2358                 flags |= FL_IS_LINEBREAK
2359             elif width >= 1 and w > width:
2360                 while w > width:
2361                     split_width = split_pos = 0
2362                     # split the word
2363                     for c in word:
2364                         cw = self._get_text_width(
2365                             c, self.tab_width, self._label_cached
2366                         )
2367                         if split_width + cw > width:
2368                             break
2369                         split_width += cw
2370                         split_pos += 1
2371                     if split_width == split_pos == 0:
2372                         # can't fit the word in, give up
2373                         break
2374                     lines_append(word[:split_pos])
2375                     lines_flags_append(flags)
2376                     flags = FL_IS_WORDBREAK
2377                     word = word[split_pos:]
2378                     w -= split_width
2379                 x = w
2380                 line.append(word)
2381             else:
2382                 x += w
2383                 line.append(word)
2384         if line or flags & FL_IS_LINEBREAK:
2385             lines_append(_join(line))
2386             lines_flags_append(flags)

2747     def _set_cursor(self, pos):
2748         if not self._lines:
2749             self._trigger_refresh_text()
2750             return
2751         l = self._lines
2752         cr = boundary(pos[1], 0, len(l) - 1)
2753         cc = boundary(pos[0], 0, len(l[cr]))
2754         cursor = cc, cr
2755 
2756         # adjust scrollview to ensure that the cursor will be always inside our
2757         # viewport.
2758         padding_left = self.padding[0]
2759         padding_right = self.padding[2]
2760         viewport_width = self.width - padding_left - padding_right
2761         sx = self.scroll_x
2762         offset = self.cursor_offset()
2763 
2764         # if offset is outside the current bounds, readjust
2765         if offset > viewport_width + sx:
2766             self.scroll_x = offset - viewport_width
2767         if offset < sx:
2768             self.scroll_x = offset
2769 
2770         # do the same for Y
2771         # this algo try to center the cursor as much as possible
2772         dy = self.line_height + self.line_spacing
2773         offsety = cr * dy
2774         sy = self.scroll_y
2775         padding_top = self.padding[1]
2776         padding_bottom = self.padding[3]
2777         viewport_height = self.height - padding_top - padding_bottom - dy
2778         if offsety > viewport_height + sy:
2779             sy = offsety - viewport_height
2780         if offsety < sy:
2781             sy = offsety

3150         flags = self._lines_flags
3151         lines = self._lines
3152         len_lines = len(lines)
3153         less_flags = len(flags) < len_lines
3154         if less_flags:
3155             flags.append(1)
3156         text = ''.join(
3157             ('\n' if (flags[i] & FL_IS_LINEBREAK) else '') + lines[i]
3158             for i in range(len_lines)

3160         if less_flags:
3161             flags.pop()
3162         return text
